Класс NumericAwarePrefix

Описание

Внутренний класс, содержащий префиксы для обозначения числовых типов данных в строковых представлениях. Используется для корректной обработки числовых значений в механизмах сериализации/десериализации или парсинга данных.

Константы

Константа Значение Описание
NUMERIC_PREFIX_INT_32 " " Префикс для 32-битных целых чисел (соответствует символу пробела в HTML-сущностях)
NUMERIC_PREFIX_INT_64 "@" Префикс для 64-битных целых чисел (соответствует символу "@" в HTML-сущностях)
NUMERIC_PREFIX_DOUBLE "&#d;" Префикс для чисел с плавающей точкой двойной точности (нестандартная HTML-сущность)

Особенности

  • Класс имеет модификатор internal - доступен только внутри текущей сборки SDK
  • Префиксы используются для внутренней маршрутизации числовых типов при обработке данных
  • Значения представлены в виде HTML-сущностей для обеспечения безопасной передачи в различных контекстах

Примеры использования

Формирование строкового представления числа


// Внутреннее использование внутри SDK
string FormatNumber(int value) 
{
    return $"{NumericAwarePrefix.NUMERIC_PREFIX_INT_32}{value}";
}

Парсинг числового значения


if (input.StartsWith(NumericAwarePrefix.NUMERIC_PREFIX_DOUBLE)) 
{
    double result = ParseDouble(input.Substring(3));
}
Важно:
  • Префикс NUMERIC_PREFIX_DOUBLE содержит нестандартную HTML-сущность "&#d;" - требует внимания при парсинге
  • Класс предназначен для внутреннего использования - не рекомендуется ссылаться на него напрямую в клиентском коде
  • При сериализации/десериализации используйте стандартные методы SDK для работы с числовыми типами
