Класс TaskFields [Устаревший]

Важно: Этот класс помечен как устаревший. Рекомендуется использовать обновленные API для работы с задачами.

Описание

Статический класс для создания условий поиска задач через интерфейс ISearchQueryBuilder. Содержит предопределенные поля для фильтрации и поиска объектов типа "Задача".

Поля

Свойство Тип поля Идентификатор Описание
Id GuidField "DObject.Id" Уникальный идентификатор задачи
InitiatorPositionId Int32Field "Initiator CEA10431..." Идентификатор должности инициатора
ExecutorPositionId Int32Field "Executor 7620B1D4..." Идентификатор должности исполнителя
CreatedDate DateTimeField "DObject.Created" Дата создания задачи
State EnumField "TaskState B65D6C5B..." Состояние задачи (см. State перечисление)
Title StringField "Title 578C65E3..." Заголовок задачи
Description StringField "Description 6589B3AC..." Описание задачи
DeadlineDate DateTimeField "TaskDeadlineDate DE08B9D6..." Срок выполнения задачи
DateOfAssignment DateTimeField "DateOfAssignment B8A06EEF..." Дата назначения задачи
DateOfCompletion DateTimeField "DateOfCompletion_C8FEC8EC..." Дата завершения задачи
DateOfStart DateTimeField "DateOfStart_697B8750..." Дата начала выполнения
AllText StringField "DObject.AllText" Полнотекстовый поиск по задаче
TaskIsVersion BoolField "TaskIsVersion_EECBDB6A..." Признак версионности задачи
Kind EnumField "TaskKind_A87C8C5E..." Тип задачи (см. TaskKind)
Tag StringField "Tags_BCA19031..." Теги задачи

Особенности

  • Использует Int32To64Field для преобразования 32-битных идентификаторов в 64-битные
  • Поля State и Kind работают с перечислениями через EnumAsInt64Field
  • Содержит ссылки на устаревшие атрибуты из SystemAttributeNames

Примеры использования

Поиск задач по статусу


var query = SearchQueryBuilder.Create()
    .Where(TaskFields.State.EqualTo(State.Completed))
    .Execute();

Фильтрация по дате дедлайна


var overdueTasks = SearchQueryBuilder.Create()
    .Where(TaskFields.DeadlineDate.LessThan(DateTime.Now))
    .ToList();
Важно:
  • Избегайте использования в новых проектах - класс устарел
  • Для полнотекстового поиска используйте AllText.Contains("текст")
  • Проверяйте актуальность перечислений State и TaskKind
