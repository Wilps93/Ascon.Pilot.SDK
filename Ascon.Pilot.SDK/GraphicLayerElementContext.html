Класс GraphicLayerElementContext

Описание

Контекст для работы с элементами графических слоев. Наследуется от MarshalByRefObject для поддержки распределенных операций и сериализации.

Наследование


System.Object
    System.MarshalByRefObject
        GraphicLayerElementContext

Конструктор


public GraphicLayerElementContext(
    IDataObject dataObject,
    Guid elementId,
    Guid contentId
)

Инициализирует контекст с обязательными параметрами.

Параметры конструктора

Параметр Тип Описание
dataObject IDataObject Целевой объект, к которому относится графический элемент
elementId Guid Идентификатор файла элемента графического слоя
contentId Guid Идентификатор файла содержимого графического слоя

Свойства

Свойство Тип Доступ Описание
DataObject IDataObject Только чтение Объект, связанный с графическим элементом
ElementId Guid Только чтение Уникальный идентификатор элемента слоя
ContentId Guid Только чтение Уникальный идентификатор содержимого слоя

Примеры использования

Создание контекста


IDataObject targetObject = GetTargetDataObject();
Guid elementGuid = Guid.NewGuid();
Guid contentGuid = Guid.NewGuid();

GraphicLayerElementContext context = new GraphicLayerElementContext(
    targetObject,
    elementGuid,
    contentGuid
);

Проверка идентификаторов


if (context.ElementId == Guid.Empty)
{
    // Обработка ошибки: неинициализированный идентификатор элемента
    throw new InvalidOperationException("ElementId не задан");
}
Важно:
  • Свойства DataObject, ElementId и ContentId неизменяемы после создания
  • Всегда проверяйте идентификаторы на Guid.Empty перед использованием
  • Для корректной работы в распределенных системах используйте MarshalByRefObject возможности
Рекомендации:
  • Храните контекст в сериализуемых структурах при работе с удаленными вызовами
  • Используйте Guid.NewGuid() для генерации уникальных идентификаторов
  • Для отладки сохраняйте идентификаторы в логах
