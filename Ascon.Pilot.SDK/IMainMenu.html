Интерфейс IMainMenu

Назначение

Позволяет расширять главное меню приложения, добавляя пользовательские пункты меню. Наследует базовый функционал контекстных меню из IBaseContextMenu.

Методы интерфейса

Метод Параметры Описание
BuildMenu menuHost: IMenuHost Вызывается перед отображением меню. Используется для динамического формирования элементов меню

Пример реализации


public class CustomMainMenu : IMainMenu 
{
    public void BuildMenu(IMenuHost menuHost)
    {
        // Создаем новый пункт меню
        var menuItem = new MenuItem("Custom Action", delegate 
        {
            // Логика обработки нажатия
        });
        
        // Добавляем пункт в главное меню
        menuHost.AddMenuItem(menuItem);
    }
}

Ключевые особенности

  • Метод BuildMenu вызывается каждый раз перед отображением меню
  • Позволяет динамически изменять содержимое меню в зависимости от контекста
  • Используйте IMenuHost для управления элементами меню
Важно: Для корректной работы необходимо зарегистрировать реализацию интерфейса в системе через механизм расширений SDK
