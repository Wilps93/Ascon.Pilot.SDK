Интерфейс INewTabPageHost

Назначение

Предоставляет методы для добавления пользовательских кнопок на страницу новой вкладки приложения. Используется в связке с интерфейсом INewTabPage для расширения функционала вкладок.

Методы

Метод Параметры Описание
AddButton
  • title (string) - Заголовок расширения (используется как текст кнопки и заголовок вкладки)
  • name (string) - Уникальное имя кнопки
  • toolTip (string) - Текст всплывающей подсказки
  • svgIcon (byte[]) - Иконка в формате SVG
 Добавляет новую кнопку на страницу новой вкладки

Пример использования


public void BuildNewTabPage(INewTabPageHost host)
{
    // Чтение SVG-иконки из файла
    byte[] iconData = File.ReadAllBytes("custom_icon.svg");
    
    // Добавление кнопки
    host.AddButton(
        title: "Мое расширение",
        name: "custom_extension_button",
        toolTip: "Создать специальный объект",
        svgIcon: iconData
    );
}

Ключевые особенности

  • Иконки должны быть в формате SVG (рекомендуемый размер: 48x48px)
  • Параметр name должен быть уникальным в рамках приложения
  • Заголовок (title) отображается как на кнопке, так и в заголовке вкладки
  • Метод вызывается в реализации INewTabPage.BuildNewTabPage
Важно:
  • Убедитесь, что SVG-файл корректно отображается в интерфейсе
  • Для динамических иконок используйте svgIcon вместо растровых форматов
  • Избегайте дублирования имен кнопок для предотвращения конфликтов
