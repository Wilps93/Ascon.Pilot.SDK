Интерфейс ITaskObject

Важно!
  • Интерфейс помечен как устаревший ([Obsolete])
  • Объект автоматически уничтожается через 2 минуты
  • Любые обращения к свойствам/методам после уничтожения вызовут исключение

Описание

Контейнер для временного хранения данных задачи с метаданными. Предназначен для безопасной передачи информации между компонентами системы.

Методы

Метод Описание
ToDataObject() Конвертирует ITaskObject в IDataObject

Пример использования


public void ProcessTask(ITaskObject task)
{
    // Проверка состояния данных
    if (task.DataState != DataState.Valid)
        throw new InvalidOperationException("Объект устарел");

    // Создание безопасной копии данных
    var taskCopy = new TaskCopy
    {
        Id = task.Id,
        Title = task.Title,
        Deadline = task.DeadlineDate,
        Attributes = new Dictionary(task.Attributes)
    };

    // Работа с данными
    Console.WriteLine($"Обработка задачи: {taskCopy.Title}");
}

public class TaskCopy
{
    public Guid Id { get; set; }
    public string Title { get; set; }
    public DateTime Deadline { get; set; }
    public IDictionary Attributes { get; set; }
}

Рекомендации

Важно:
  • Не сохраняйте ссылки на объект дольше 2 минут
  • Делайте глубокие копии нужных свойств (пример выше)
  • Всегда проверяйте DataState перед использованием
  • Используйте ToDataObject() для конвертации в постоянный объект
  • Для работы с IsAgreementTask используйте альтернативные методы
Известные проблемы:
  • В XML-комментариях опечатка: "iniator" вместо "initiator"
  • Отсутствует метод удаления задач - используйте ITaskModifier
