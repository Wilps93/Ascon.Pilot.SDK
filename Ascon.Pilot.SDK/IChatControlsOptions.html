Интерфейс IChatControlsOptions

Fluent-интерфейс: Все методы возвращают IChatControlsOptions, позволяя использовать цепочки вызовов (method chaining).

Описание

Предоставляет настройки отображения элементов управления чатом. Позволяет конфигурировать видимость различных UI-компонентов.

Методы

Метод Параметр Описание
WithShowChatInfo value (bool) Управляет отображением информации о чате (например, участники, дата создания)
WithShowTabHeaders value (bool) Управляет видимостью заголовков вкладок в интерфейсе чата
WithShowObjectCaption value (bool) Управляет отображением заголовков объектов в чате

Пример использования


var chatOptions = new ChatOptions()
    .WithShowChatInfo(true)
    .WithShowTabHeaders(false)
    .WithShowObjectCaption(true);

// Применение настроек к контролу чата
chatControl.ApplyOptions(chatOptions);
Рекомендации:
  • Используйте цепочки вызовов для компактной настройки
  • Значения по умолчанию зависят от реализации
  • Методы не изменяют существующий экземпляр, а возвращают новый с обновленными настройками
Важно: Интерфейс является частью системы конфигурации и не содержит логики отображения - реализация должна быть предоставлена в конкретном классе.
