Перечисление NotificationTrigger

Описание

Определяет типы триггеров уведомлений в Pilot-Storage. Используется для определения причины генерации уведомления.

Значения

Значение Числовой код Описание
Loading 0 Уведомление сгенерировано при загрузке Pilot-Storage
Update 1 Файл или директория были обновлены

Примеры использования

Обработка триггеров


void HandleNotification(NotificationTrigger trigger)
{
    switch (trigger)
    {
        case NotificationTrigger.Loading:
            Console.WriteLine("Инициализация хранилища...");
            break;
            
        case NotificationTrigger.Update:
            Console.WriteLine("Обнаружены изменения данных");
            ProcessUpdates();
            break;
            
        default:
            throw new ArgumentOutOfRangeException(nameof(trigger), trigger, null);
    }
}

Проверка типа уведомления


if (trigger == NotificationTrigger.Loading)
{
    InitializeCache();
}
else
{
    CheckForChanges();
}
Примечание: Значения перечисления не являются битовыми флагами. Используйте обычные операторы сравнения для работы с ними.
