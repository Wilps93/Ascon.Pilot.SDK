Интерфейс IMouseLeftClickListener

Описание

Предоставляет механизм обработки событий клика левой кнопкой мыши в графических компонентах системы. Используется для реализации пользовательских действий при взаимодействии с визуальными элементами.

Методы

Метод Параметры Описание
OnLeftMouseButtonClick pointContext: XpsRenderClickPointContext Вызывается при клике левой кнопкой мыши. Содержит контекст события с информацией о позиции и состоянии

Пример реализации


public class CustomClickListener : IMouseLeftClickListener 
{
    public void OnLeftMouseButtonClick(XpsRenderClickPointContext pointContext)
    {
        // Получение координат клика
        var position = pointContext.Position;
        
        // Проверка наличия элемента под курсором
        if (pointContext.HitTestResult != null)
        {
            // Реакция на клик по визуальному элементу
            Console.WriteLine($"Клик на элемент: {pointContext.HitTestResult.Visual.Name}");
        }
    }
}

Ключевые особенности

  • Контекст XpsRenderClickPointContext содержит:
    • Координаты клика (Position)
    • Результат хит-теста (HitTestResult)
    • Состояние модификаторов клавиатуры
  • Должен быть подписан на события соответствующего визуального компонента
  • Реализация должна учитывать многопоточность при работе с UI
Важно: Убедитесь, что объект реализующий интерфейс, правильно подписан на события целевого компонента и своевременно удаляется при уничтожении элемента
