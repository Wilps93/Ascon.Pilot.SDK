Перечисление GraphicLayerOption

Описание

Определяет режимы внедрения графических слоев, текстовых меток и штрих-кодов в файлы. Используется в классе FileSaverOptions для настройки сохранения файлов.

Значения

Значение Описание
InjectFloatingRelatable Внедрять элементы с учетом состояния параметра IsFloating. Плавающие элементы сохраняются отдельно
ForceInject Принудительно внедрять все графические элементы, игнорируя настройки
KeepAllAsFloating Сохранять все графические элементы как плавающие, даже если они привязаны к объектам
LoseGraphicLayers Удалять все графические слои и метки из файла

Примеры использования

Настройка сохранения файла


FileSaverOptions options = new FileSaverOptions
{
    GraphicLayerOption = GraphicLayerOption.ForceInject
};

fileSaver.Save(document, options);

Условная обработка


switch (options.GraphicLayerOption)
{
    case GraphicLayerOption.LoseGraphicLayers:
        RemoveAllGraphics(document);
        break;
    case GraphicLayerOption.KeepAllAsFloating:
        ConvertToFloatLayers(document);
        break;
}
Важно:
  • Значение LoseGraphicLayers необратимо удаляет все графические элементы
  • В описании перечисления присутствует опечатка: "gragraphic" вместо "graphic"
  • KeepAllAsFloating может нарушить привязки элементов к объектам
Рекомендации:
  • Используйте InjectFloatingRelatable для сохранения структуры документа
  • Проверяйте поддержку режимов в вашей версии SDK
  • Для архивных копий используйте LoseGraphicLayers для уменьшения размера файла
