Класс SystemObjectIds

Описание

Содержит предопределенные идентификаторы системных объектов Pilot.SDK. Используется для ссылок на корневые объекты различных типов в хранилище данных.

Поля

Имя Значение (GUID) Описание
RootObjectId 00000001-0001-0001-0001-000000000001 Корневой объект всей системы
TaskRootObjectId ADB79734-723C-4FF8-8DA8-8F52E140FBF4 Корневой объект для задач
ExtensionRootObjectId E6519D37-1984-407E-96A0-1CD371F68F16 Корневой объект для расширений
ReportRootObjectId 7DAB217D-6E06-4C2C-AB77-B5EC9361415D Корневой объект для отчетов
TaskTemplateRootObjectId DB4AB44C-B3D1-4F5A-B049-13CDF0C2EFE7 Корневой объект для шаблонов задач

Особенности

  • Идентификаторы являются константами и не должны изменяться
  • Используются как точки входа для навигации по хранилищу
  • Предоставляют унифицированный доступ к системным объектам

Примеры использования

Проверка корневого объекта


Guid objectId = GetObjectId();
if (objectId == SystemObjectIds.RootObjectId)
{
    Console.WriteLine("Это корневой объект системы");
}

Работа с шаблонами задач


// Получение корневого объекта шаблонов
var templateRoot = storage.GetObject(SystemObjectIds.TaskTemplateRootObjectId);

// Поиск дочерних шаблонов
foreach (var template in templateRoot.Children)
{
    ProcessTemplate(template);
}
Важно:
  • Не используйте эти идентификаторы для создания новых объектов
  • Для проверки типа объекта используйте IsRoot() из соответствующих API
  • Избегайте прямого сравнения GUID - используйте метод Equals()
