Класс StorageContext

Важно: Название "pilot-strage" в описании класса содержит опечатку. Предположительно должно быть "pilot-storage". Уточните в актуальной документации.

Описание

Контекст для работы с хранилищем данных. Наследуется от MarshalByRefObject, что позволяет использовать объект в сценариях удаленного взаимодействия и междоменной коммуникации.

Конструктор


public StorageContext(IEnumerable selectedObjects)

Инициализирует новый экземпляр контекста с выбранными объектами хранилища.

Параметры

Параметр Тип Описание
selectedObjects IEnumerable Коллекция выбранных объектов хранилища

Свойства

Свойство Тип Доступ Описание
SelectedObjects IEnumerable Только чтение Коллекция объектов хранилища, переданных в конструкторе

Особенности

  • Свойство SelectedObjects инициализируется один раз при создании объекта
  • Наследование от MarshalByRefObject позволяет использовать контекст в распределенных системах
  • Отсутствие публичного сеттера гарантирует неизменяемость коллекции после создания

Пример использования


// Получение выбранных объектов из хранилища
IEnumerable selected = GetSelectedObjects();

// Создание контекста
StorageContext context = new StorageContext(selected);

// Работа с объектами
foreach (var obj in context.SelectedObjects)
{
    ProcessObject(obj);
}
Рекомендации:
  • Для изменения списка объектов создавайте новый экземпляр контекста
  • Убедитесь в поддержке сериализации объектов при использовании в распределенных системах
