Перечисление AttributeUniquenessType

Описание

Определяет уровень контроля уникальности значений атрибутов объектов.

Значения

Значение Описание
NotUnique Допускаются неуникальные значения атрибута
Warning Предупреждение при создании неуникального значения
Prohibition Запрет создания дублирующихся значений

Примеры использования

Проверка уровня уникальности


AttributeUniquenessType uniqueness = GetUniquenessType();
switch (uniqueness)
{
    case AttributeUniquenessType.NotUnique:
        Console.WriteLine("Проверка уникальности не требуется");
        break;
    case AttributeUniquenessType.Warning:
        ShowWarning("Обнаружены дубликаты!");
        break;
    case AttributeUniquenessType.Prohibition:
        throw new InvalidOperationException("Дублирование запрещено");
}

Применение при создании атрибута


IAttributeDefinition attrDef = metadata.CreateAttribute(
    "UserLogin",
    AttributeType.String,
    AttributeUniquenessType.Prohibition
);
Важно:
  • Prohibition вызывает исключение при попытке сохранения дубликата
  • Warning позволяет сохранить значение, но уведомляет пользователя
  • Используется в связке с IAttributeDefinition при настройке метаданных

Связь с системой

Контроль уникальности реализуется через:

  • Механизмы валидации данных при сохранении объектов
  • Проверку целостности данных в БД
  • Интерфейсные уведомления для пользователей
