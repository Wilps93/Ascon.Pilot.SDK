Класс PilotDialogServiceExtensions

Описание

Статический класс с методами расширения для IPilotDialogService. Предоставляет удобный способ отображения диалогов выбора организационных единиц.

Методы расширения

ShowOrganisationUnitSelectorDialog()

public static IEnumerable ShowOrganisationUnitSelectorDialog(this IPilotDialogService pilotDialogService, IPilotDialogOptions options = null)
Параметр Тип Описание
pilotDialogService IPilotDialogService Сервис диалоговых окон
options IPilotDialogOptions Опции диалога (необязательный параметр)

Возвращаемое значение

IEnumerable - Коллекция выбранных организационных единиц

Примеры использования

Базовый вызов без опций


var selectedUnits = dialogService.ShowOrganisationUnitSelectorDialog();
foreach (var unit in selectedUnits)
{
    // Обработка выбранных единиц
    ProcessUnit(unit);
}

Вызов с настройкой опций


var options = new PilotDialogOptions
{
    Title = "Выбор подразделения",
    AllowMultipleSelection = true,
    ShowGroups = false
};

var departments = dialogService.ShowOrganisationUnitSelectorDialog(options);
Важно:
  • Метод является оберткой над рефлексивным вызовом Extensions.InvokeMethod()
  • Поддерживает выбор всех типов организационных единиц (Department, Position, Group)
  • Возвращаемая коллекция может быть пустой, если пользователь отменил выбор
Рекомендации:
  • Используйте options для настройки фильтров и поведения диалога
  • Проверяйте возвращаемое значение на null перед использованием
  • Для работы с результатом используйте методы из OrganisationUnitExtensions
