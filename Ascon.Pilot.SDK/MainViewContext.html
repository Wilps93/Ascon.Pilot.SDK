Класс MainViewContext

Важно:
  • Базовый класс для контекстов главного интерфейса
  • В текущей версии не содержит пользовательских данных

Назначение

Предоставляет контекстную информацию для работы с главным представлением системы. Используется как маркерный класс для инфраструктурных механизмов.

Наследование


System.Object 
→ System.MarshalByRefObject 
→ Ascon.Pilot.SDK.MainViewContext

Особенности реализации

  • Наследуется от MarshalByRefObject для поддержки удаленного взаимодействия
  • Предназначен для расширения в будущих версиях SDK
  • Может использоваться в качестве базового класса для специфичных контекстов

Пример использования


public void ConfigureMainView(MainViewContext context)
{
    // Передача контекста в инфраструктурные методы
    _viewService.Initialize(context);
    
    // Для будущих расширений:
    // if (context is IExtendedViewContext extended)
    // {
    //     extended.ApplyCustomSettings();
    // }
}
Внимание:
  • Не содержит функциональных методов в текущей реализации
  • Не предназначен для прямого наследования пользовательскими классами
Рекомендации:
  • Используйте для интеграции с системными компонентами интерфейса
  • Проверяйте наличие расширений через оператор is в будущих версиях
