Класс DataObjectChange

Описание

Класс инкапсулирующий изменение состояния объекта. Содержит снимки объекта до и после модификации.

Иерархия наследования

  • System.Object
  • Ascon.Pilot.SDK.DataObjectChange

Конструктор


internal DataObjectChange(IDataObject old, IDataObject @new)

Инициализирует новый экземпляр изменения объекта. Доступен только внутри сборки.

Параметр Описание
old Состояние объекта до изменения
new Состояние объекта после изменения

Свойства

Свойство Тип Описание
Old IDataObject Объект до изменений (только для чтения)
New IDataObject Объект после изменений (только для чтения)

Примеры использования

Анализ изменений объекта


DataObjectChange change = GetObjectChange();
if (change.Old.Version != change.New.Version)
{
    Console.WriteLine($"Обнаружено изменение версии: {change.Old.Version} -> {change.New.Version}");
}

Сравнение состояний


void LogChanges(DataObjectChange change)
{
    // Сравнение свойств объектов
    foreach (var attr in change.New.Attributes)
    {
        if (!change.Old.Attributes[attr.Key].Equals(attr.Value))
        {
            Console.WriteLine($"Изменен атрибут {attr.Key}: {change.Old.Attributes[attr.Key]} -> {attr.Value}");
        }
    }
}
Важно:
  • Класс предназначен только для чтения - свойства Old и New нельзя изменить после создания
  • Используется в механизмах отслеживания изменений и аудита
  • Доступ к конструктору ограничен внутренней реализацией SDK

Связь с системой

Интегрирован с:

  • Системой версионирования объектов
  • Механизмами событий обновления данных
  • Аудит-логами изменений
