Класс UnloadedEventArgs

Описание

Событийные аргументы, уведомляющие о завершении выгрузки расширения из системы. Наследуется от MarshalByRefObject для поддержки работы в распределенных средах.

Наследование

  • System.MarshalByRefObject
  • → UnloadedEventArgs

Использование

Используется в событиях, связанных с жизненным циклом модулей/расширений:

  • Подписка на событие выгрузки
  • Оповещение о завершении работы компонента
  • Очистка ресурсов

Примеры

Подписка на событие


public class ExtensionManager 
{
    public event EventHandler<UnloadedEventArgs> Unloaded;

    protected virtual void OnUnloaded()
    {
        Unloaded?.Invoke(this, new UnloadedEventArgs());
    }
}

Обработка события


// Подписка на событие выгрузки
extensionManager.Unloaded += (sender, args) => 
{
    // Выполняем завершающие действия
    Console.WriteLine("Расширение выгружено из системы");
};
Важно:
  • Класс не содержит дополнительных данных - служит маркером события
  • Наследование от MarshalByRefObject позволяет использовать экземпляры в удаленных вызовах
  • Рекомендуется использовать в паре с LoadedEventArgs для полного цикла управления

Особенности

  • Не требует реализации дополнительных методов
  • Может использоваться как базовый класс для кастомных аргументов выгрузки
  • Поддерживает стандартные паттерны работы с событиями .NET
