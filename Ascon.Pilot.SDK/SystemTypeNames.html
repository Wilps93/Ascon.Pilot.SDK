Класс SystemTrayContext

Внимание:
  • Класс не содержит публичных членов в текущей версии SDK
  • В описании класса присутствует опечатка: "Contex" вместо "Context"

Описание

Базовый класс для работы с контекстом системного трея. Наследуется от MarshalByRefObject, что позволяет использовать объект в сценариях удаленного взаимодействия и междоменной коммуникации.

Особенности

  • Предназначен для расширения в производных классах
  • Может использоваться как контейнер данных для операций с системным трейем
  • Наследование от MarshalByRefObject позволяет передавать объект между доменами приложений

Пример использования


// Создание экземпляра контекста
SystemTrayContext context = new SystemTrayContext();

// Передача контекста в метод API
trayService.ProcessContext(context);
Рекомендации:
  • Проверьте документацию последней версии SDK - возможно, класс был расширен
  • Используйте как маркерный объект для операций с системным трейем
  • Для кастомизации создавайте производные классы с дополнительными свойствами
