Интерфейс IStageObject

Внимание! Этот интерфейс помечен как устаревший (Obsolete) и будет удален в будущих версиях.
Временный объект: Экземпляр этого интерфейса автоматически уничтожается через 2 минуты. Попытки обращения к свойствам после этого вызовут исключение.

Описание

Представляет временный контейнер для передачи данных этапа (stage) и его атрибутов. Используется в связке с устаревшим интерфейсом IStageChangeBuilder.

Свойства

Свойство Тип Описание
Id Guid Уникальный идентификатор этапа
Order int Порядковый номер этапа в рабочем процессе

Пример использования (устаревший)


// Пример legacy-кода (не рекомендуется)
var stageBuilder = stageChangeBuilderFactory.Create();
stageBuilder.SetStageOrder(3);
var stage = stageBuilder.StageObject;

// Важно: используйте данные в течение 2 минут!
ProcessStageId(stage.Id);

  • Проверьте документацию на наличие современных альтернатив
