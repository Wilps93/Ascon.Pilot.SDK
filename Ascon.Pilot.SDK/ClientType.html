Перечисление ClientType

Описание

Определяет типы клиентских приложений в системе. Используется для идентификации версий и функциональных возможностей клиента.

Значения

Значение Описание
Ice Базовая версия клиента ICE
IceEnterprise Корпоративная версия ICE с расширенными возможностями
ThreeDStorage Клиент для работы с 3D-хранилищем
Ecm Клиент для ECM-систем (Enterprise Content Management)
Bim Клиент для работы с BIM-моделями (Building Information Modeling)

Примеры использования

Определение типа клиента


ClientType clientType = GetCurrentClientType();
switch (clientType)
{
    case ClientType.IceEnterprise:
        EnableAdvancedFeatures();
        break;
    case ClientType.ThreeDStorage:
        Initialize3DViewer();
        break;
    default:
        SetupBasicInterface();
        break;
}

Проверка поддержки функционала


if (clientType == ClientType.Bim)
{
    // Активация BIM-инструментов
    BIMTools.Enable();
}
Важно:
  • Тип клиента определяется при подключении к серверу
  • IceEnterprise включает все функции базового ICE + корпоративные модули
  • ThreeDStorage и Bim могут требовать специфических лицензий

Связь с системой

Используется в:

  • Механизмах авторизации и лицензирования
  • Логике активации функциональных модулей
  • Системе настройки пользовательского интерфейса
