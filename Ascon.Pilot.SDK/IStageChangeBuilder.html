Интерфейс IStageChangeBuilder

Внимание! Этот интерфейс и все его методы помечены как устаревшие (Obsolete). Использование может привести к ошибкам в будущих версиях SDK.

Описание

Флюент-интерфейс для модификации этапов (stage) в системе. Не рекомендуется к использованию - данный функционал сохраняется только для обратной совместимости.

Устаревшие методы

Метод/Свойство Описание Статус
SetStageOrder(int order) Метод для изменения порядка этапа Obsolete
IStageObject StageObject { get; } Свойство для получения связанного этапа Obsolete

Примеры (только для legacy-кода)


// Пример устаревшего кода (не рекомендуется)
var builder = stageChangeBuilderFactory.Create();
builder.SetStageOrder(5);
IStageObject stage = builder.StageObject;
Рекомендации:
  • Избегайте использования этого интерфейса в новом коде
  • Проверьте документацию SDK на наличие современных альтернатив
  • Планируйте миграцию legacy-кода, использующего эти методы
Важно: Все методы и свойства этого интерфейса будут удалены в будущих версиях SDK без обратной совместимости.
