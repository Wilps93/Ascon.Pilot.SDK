Перечисление AutoimportSource

Описание

Определяет источник автоматически импортируемых файлов в систему.

Значения

Значение Описание
Unknown Источник файла не определен или неизвестен
UserFolder Файл импортирован из пользовательской директории автозагрузки
PilotXps Файл получен через виртуальный принтер Pilot XPS

Примеры использования

Обработка источника файла


AutoimportSource source = GetFileSource();
switch (source)
{
    case AutoimportSource.UserFolder:
        Console.WriteLine("Файл из пользовательской папки");
        break;
    case AutoimportSource.PilotXps:
        ProcessXpsDocument();
        break;
    default:
        LogUnknownSource();
        break;
}

Создание объекта с указанием источника


ImportSettings settings = new ImportSettings
{
    Source = AutoimportSource.PilotXps,
    FilePath = "C:\\Temp\\document.xps"
};
Важно:
  • Значение Unknown обычно указывает на ошибку или устаревшие данные
  • PilotXps предполагает специфическую обработку XPS-документов
  • Используется в механизмах автоматического импорта и обработки входящих файлов

Связь с системой

Интегрирован с:

  • Модулем автоматического импорта документов
  • Сервисом обработки печатных форм через Pilot XPS
  • Системой логирования операций с файлами
