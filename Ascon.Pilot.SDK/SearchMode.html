Перечисление SearchMode

Описание

Определяет режимы поиска в системе. Поддерживает комбинации флагов для гибкой настройки области поиска.

Базовые значения: Attributes (1) Files (2)

Значения перечисления

Значение Числовое представление Описание
Attributes 1 Поиск по атрибутам объектов
Files 2 Поиск по содержимому файлов
Attributes | Files 3 Комбинированный поиск по атрибутам и файлам

Примеры использования

Поиск по атрибутам


var searchMode = SearchMode.Attributes;
searchService.PerformSearch(query, searchMode);

Комбинированный поиск


var mode = SearchMode.Attributes | SearchMode.Files;
var results = searchService.Find(pattern, mode);

Проверка флагов


if ((currentMode & SearchMode.Files) == SearchMode.Files)
{
    // Включена индексация файлов
    EnableFileIndexing();
}
Важно:
  • Используйте оператор | для комбинации режимов
  • Проверяйте флаги через битовые операции &
  • Значение 0 означает отсутствие режимов (поиск не выполняется)
Рекомендации:
  • Для полнотекстового поиска используйте Files
  • Для метаданных и свойств объектов - Attributes
  • Комбинированный режим увеличивает время выполнения запроса
