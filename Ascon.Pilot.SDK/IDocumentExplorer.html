Интерфейс IDocumentExplorer

Назначение: Обеспечивает взаимодействие с компонентом DocumentExplorer для обновления отображения документов.

Описание

Предоставляет метод для программного обновления предварительного просмотра документа в интерфейсе.

Методы

Метод Описание
UpdateDocumentPreview() Принудительно обновляет отображение предварительного просмотра документа.
Используется при изменениях, которые не отслеживаются автоматически.

Примеры использования

Обновление после редактирования


public void ApplyChangesAndRefresh(IDocumentExplorer explorer)
{
    // Внесение изменений в документ
    ModifyDocumentData();
    
    // Обновление UI
    explorer.UpdateDocumentPreview();
}

Обработка события обновления


documentSavedEvent += (sender, args) =>
{
    // После сохранения документа
    _documentExplorer.UpdateDocumentPreview();
};
Рекомендации:
  • Вызывайте метод только при необходимости - избыточные вызовы могут снизить производительность
  • Убедитесь, что DocumentExplorer инициализирован и доступен
  • Для массовых изменений используйте пакетные обновления
Важно:
  • Метод должен вызываться в UI-потоке
  • Не обновляет данные документа - только визуальное представление
  • Не заменяет необходимость сохранения изменений
