Интерфейс IBaseContextMenu

Описание

Базовый интерфейс для реализации обработчиков контекстных меню. Позволяет реагировать на действия пользователя при выборе пунктов меню.

Методы

Метод Параметры Описание
OnMenuItemClick
  • itemName (string) - идентификатор или название выбранного пункта меню
 Вызывается при клике пользователя на пункт контекстного меню

Пример реализации


public class CustomContextMenu : IBaseContextMenu
{
    public void OnMenuItemClick(string itemName)
    {
        switch (itemName)
        {
            case "copy":
                ClipboardService.Copy();
                break;
            case "delete":
                DeleteSelectedItem();
                break;
            default:
                Logger.Warn($"Неизвестный пункт меню: {itemName}");
                break;
        }
    }
}
Рекомендации:
  • Всегда обрабатывайте неизвестные itemName для избежания ошибок
  • Используйте осмысленные имена пунктов меню (например, "edit_copy", "view_zoom")
  • Для сложной логики используйте паттерн Command или делегаты
Важно: Реализация должна быть потокобезопасной, если обработка выполняется асинхронно.
