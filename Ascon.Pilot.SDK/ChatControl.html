Класс ChatControl

Описание

Класс для интеграции элементов управления чатом в UI-компоненты. Наследуется от MarshalByRefObject для поддержки кросс-доменного взаимодействия.

Иерархия наследования

  • System.Object
  • System.MarshalByRefObject
  • Ascon.Pilot.SDK.ChatControl

Конструктор


public ChatControl(FrameworkElement view, IChatControlsInteraction interaction)

Инициализирует новый экземпляр элемента управления чатом.

Параметр Описание
view Визуальный элемент управления (UI-компонент)
interaction Объект для обработки взаимодействий с чатом

Свойства

Свойство Тип Описание
View FrameworkElement Визуальное представление элемента управления
Interaction IChatControlsInteraction Обработчик взаимодействий с чатом

Примеры использования

Создание элемента управления чатом


FrameworkElement chatView = CreateChatUI();
IChatControlsInteraction interaction = new ChatInteractionHandler();
ChatControl chatControl = new ChatControl(chatView, interaction);

Интеграция с интерфейсом


public IView CreateChatPanel()
{
    ChatControl control = GetChatControl();
    // Добавление в UI
    mainGrid.Children.Add(control.View);
    // Настройка взаимодействий
    control.Interaction.SendMessage += OnSendMessage;
    return control.View;
}
Важно:
  • Свойства View и Interaction доступны только для чтения после создания объекта
  • Для корректной работы необходимо реализовать интерфейс IChatControlsInteraction
  • Используется в связке с механизмами отображения UI и обработки событий чата

Связь с системой

Взаимодействует с:

  • UI-фреймворком (WPF) через FrameworkElement
  • Системой обработки сообщений через IChatControlsInteraction
  • Механизмами маршалинга при кросс-доменном взаимодействии
