Перечисление DataState

Описание

Определяет состояние загрузки данных объекта в системе. Используется для отслеживания текущего статуса объекта в процессе работы с данными.

Значения

Значение Описание
Unknown Начальное состояние. Информация о загрузке не определена
NoData Объект не был закэширован и в данный момент загружается с сервера
Loaded Объект успешно загружен и доступен для работы
NonExistent Объект отсутствует на сервере (удален или не существовал)

Примеры использования

Проверка состояния объекта


DataState state = GetDataState();
switch (state)
{
    case DataState.Unknown:
        Console.WriteLine("Состояние не определено");
        break;
    case DataState.NoData:
        ShowLoadingIndicator();
        break;
    case DataState.Loaded:
        ProcessData();
        break;
    case DataState.NonExistent:
        ShowNotFoundError();
        break;
}

Обработка отсутствующего объекта


if (dataObject.State == DataState.NonExistent)
{
    // Логика для несуществующего объекта
    Logger.Warn("Попытка доступа к несуществующему объекту");
}
Важно:
  • Состояние NoData требует повторной попытки загрузки или отображения индикатора прогресса
  • NonExistent не всегда означает ошибку - может указывать на корректное удаление объекта
  • Используйте проверку состояния перед выполнением операций с объектом

Связь с системой

Интегрирован с:

  • Механизмами кэширования данных
  • Системой асинхронной загрузки объектов
  • Обработчиками ошибок и валидации
  • Интерфейсными компонентами для отображения состояния
