Класс XpsMergerContext

Описание

Контекст для управления процессом объединения представлений (XPS-мерджер). Наследуется от MarshalByRefObject для поддержки работы в распределенных средах.

Наследование

  • System.MarshalByRefObject
  • → XpsMergerContext

Сценарии использования

  • Передача контекстных данных между компонентами системы
  • Хранение состояния процесса объединения
  • Интеграция с механизмами рендеринга XPS-документов

Примеры

Создание контекста


var context = new XpsMergerContext();
// Передача контекста в обработчик
documentProcessor.MergeDocuments(context);

Расширение функционала


public class CustomMergerContext : XpsMergerContext
{
    public string CustomProperty { get; set; }
    
    public void Initialize()
    {
        // Инициализация расширенного контекста
    }
}
Важно:
  • В текущей реализации класс является пустым контейнером - требует расширения
  • Наследование от MarshalByRefObject позволяет использовать экземпляры в cross-appdomain сценариях
  • Рекомендуется добавлять кастомные свойства через наследование

Рекомендации

  • Используйте контекст для передачи метаданных между этапами объединения
  • Добавляйте пользовательские свойства через наследование при необходимости
  • Проверяйте доступность контекста в асинхронных операциях
