Интерфейс IPosition

Описание

Представляет должность/роль, которую занимает сотрудник в организационной структуре. Используется в связке с интерфейсом IPerson для описания иерархии позиций.

Свойства

Свойство Тип Описание
Order int Порядок сортировки позиции в рамках организационной структуры
Position int Уникальный числовой идентификатор позиции

Примеры использования

Получение информации о позиции


IPosition position = person.MainPosition;
if (position != null)
{
    Console.WriteLine($"ID позиции: {position.Position}");
    Console.WriteLine($"Порядок сортировки: {position.Order}");
}

Сравнение позиций


bool IsHigherPosition(IPosition pos1, IPosition pos2)
{
    return pos1.Order < pos2.Order;
}
Примечания:
  • Order определяет иерархию: меньшее значение = более высокая позиция
  • Position соответствует ID в базе данных организационных единиц
Важно:
  • MainPosition в IPerson может быть null - всегда проверяйте перед использованием
  • Значения Order имеют смысл только в рамках одной организационной ветки
