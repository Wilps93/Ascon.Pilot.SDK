Класс ActiveTabChangedEventArgs

Описание

Событие, уведомляющее о смене активной вкладки. Наследуется от MarshalByRefObject для поддержки маршалинга между доменами приложений.

Иерархия наследования

  • System.Object
  • System.MarshalByRefObject
  • Ascon.Pilot.SDK.ActiveTabChangedEventArgs

Конструктор


public ActiveTabChangedEventArgs(Guid activeTabId)

Инициализирует новый экземпляр класса с указанием идентификатора активной вкладки.

Параметр Описание
activeTabId Уникальный идентификатор активной вкладки (тип Guid)

Свойства

Свойство Тип Описание
ActiveTabId Guid Получает идентификатор активной вкладки. Значение доступно только для чтения после инициализации.

Примеры использования

Обработка события смены вкладки


void OnActiveTabChanged(object sender, ActiveTabChangedEventArgs e)
{
    Guid currentTabId = e.ActiveTabId;
    // Логика обработки смены вкладки
    Console.WriteLine($"Активная вкладка изменена: {currentTabId}");
}

Создание экземпляра


Guid newTabId = Guid.NewGuid();
ActiveTabChangedEventArgs args = new ActiveTabChangedEventArgs(newTabId);
Особенности:
  • Свойство ActiveTabId доступно только для чтения после инициализации
  • Наследование от MarshalByRefObject позволяет использовать объект в сценариях удаленного взаимодействия
  • Рекомендуется использовать в связке с событиями интерфейса (например, смена активного окна/вкладки)
