Класс LinkedTasksContext

Внимание! Этот класс устарел. Используйте LinkedTasksContext2 для новых разработок.

Назначение

Предоставляет контекст для работы с группой связанных задач. Используется для передачи информации о выбранных задачах и режиме выполнения операций.

Особенности:
  • Наследуется от MarshalByRefObject (поддержка удаленного взаимодействия)
  • Свойства доступны только для чтения после создания

Конструктор

LinkedTasksContext

Инициализирует новый экземпляр контекста

Параметры

selectedTasks IEnumerable Список выбранных задач для обработки
isContext bool Флаг режима выполнения операции в контексте

Свойства

SelectedTasks

Коллекция задач, выбранных для операции

Тип: IEnumerable (только для чтения)

IsContext

Определяет, выполняется ли операция в контекстном режиме

Тип: bool (только для чтения)

Примеры использования

Создание контекста (устаревший способ)


// Не рекомендуется для новых проектов
var selectedTasks = new List { task1, task2 };
var context = new LinkedTasksContext(selectedTasks, isContext: false);

// Обработка задач
ProcessTasks(context.SelectedTasks);

Проверка режима выполнения


public void ExecuteOperation(LinkedTasksContext context)
{
    if (context.IsContext)
    {
        // Контекстный режим выполнения
        ExecuteInContext(context.SelectedTasks);
    }
    else
    {
        // Стандартный режим
        ExecuteStandard(context.SelectedTasks);
    }
}

Особенности

  • Задачи в SelectedTasks должны реализовывать интерфейс ITaskObject
  • Изменение коллекции SelectedTasks после создания контекста не влияет на состояние
  • Может использоваться в распределенных системах благодаря наследованию MarshalByRefObject
Рекомендации:
  • Для новых проектов используйте LinkedTasksContext2
  • Не используйте для долгосрочного хранения данных
  • Проверяйте корректность передаваемых ITaskObject
