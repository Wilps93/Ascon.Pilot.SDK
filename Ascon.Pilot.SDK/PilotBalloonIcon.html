Перечисление PilotBalloonIcon

Описание

Определяет типы иконок для всплывающих уведомлений (баллонов) в интерфейсе приложения.

Значения перечисления

Значение Описание Визуальное представление
None Без иконки
Info Информационная иконка
Warning Предупреждающая иконка
Error Иконка ошибки

Примеры использования

Отображение уведомления


var iconType = PilotBalloonIcon.Warning;
ShowBalloonMessage("Внимание", "Низкий уровень заряда", iconType);

void ShowBalloonMessage(string title, string message, PilotBalloonIcon icon)
{
    // Логика отображения уведомления с иконкой
}

Проверка типа иконки


if (icon == PilotBalloonIcon.Error)
{
    LogError("Обнаружена критическая ошибка");
}
Важно: Визуальное представление иконок может отличаться в зависимости от темы приложения и версии SDK.
Рекомендации:
  • Используйте Info для нейтральных уведомлений
  • Warning подходит для предупреждений, требующих внимания
  • Error зарезервирован для критических сообщений
  • Значение None используется для текстовых уведомлений без иконок
