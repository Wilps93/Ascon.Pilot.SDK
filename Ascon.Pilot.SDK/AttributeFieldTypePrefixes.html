Класс AttributeFieldTypePrefixes

Описание

Внутренний статический класс, содержащий префиксы типов для формирования ключей атрибутов в поисковых запросах.

Используется для генерации уникальных идентификаторов полей в формате prefix.attributeName.

Константы типов

Константа Значение Описание
TEXT "t" Префикс для текстовых атрибутов
STRING "s" Префикс для строковых атрибутов
INT32 "i32" Префикс для 32-битных целочисленных атрибутов
INT64 "i64" Префикс для 64-битных целочисленных атрибутов
DOUBLE "d" Префикс для атрибутов с плавающей точкой

Методы

Add


public static string Add(string prefix, string value)

Формирует составной ключ из префикса и названия атрибута.

Параметр Описание
prefix Префикс типа (например, "t" для текста)
value Название атрибута

Возвращает: Строка в формате prefix.value

Примеры использования

Формирование ключа для текстового атрибута


string key = AttributeFieldTypePrefixes.Add(AttributeFieldTypePrefixes.TEXT, "Description");
// Результат: "t.Description"

Создание ключа для числового атрибута


string numericKey = AttributeFieldTypePrefixes.Add("d", "Price");
// Результат: "d.Price"
Важно:
  • Класс имеет модификатор internal и не доступен за пределами сборки
  • Используется в связке с классом AttributeFields для построения поисковых условий
  • Разделение префиксов позволяет избежать конфликтов имен между разными типами атрибутов

Связь с другими компонентами

Префиксы этого класса используются в:

  • Методах класса AttributeFields для указания типов полей
  • Механизме поиска объектов через ISearchTerm
  • Системе метаданных атрибутов
