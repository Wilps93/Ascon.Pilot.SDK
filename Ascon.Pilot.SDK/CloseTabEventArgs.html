Класс CloseTabEventArgs

Описание

Класс события, уведомляющий о закрытии вкладки с указанием её заголовка и идентификатора. Наследуется от MarshalByRefObject для поддержки кросс-доменного взаимодействия.

Иерархия наследования

  • System.Object
  • System.MarshalByRefObject
  • Ascon.Pilot.SDK.CloseTabEventArgs

Конструктор


public CloseTabEventArgs(string tabTitle, Guid tabId)

Инициализирует новый экземпляр события закрытия вкладки.

Параметр Описание
tabTitle Заголовок закрытой вкладки
tabId Уникальный идентификатор закрытой вкладки (тип Guid)

Свойства

Свойство Тип Описание
TabTitle string Заголовок вкладки (доступно только для чтения)
TabId Guid Идентификатор вкладки (доступно только для чтения)

Примеры использования

Обработка события закрытия вкладки


void OnTabClosed(object sender, CloseTabEventArgs e)
{
    Console.WriteLine($"Закрыта вкладка: {e.TabTitle} (ID: {e.TabId})");
    // Логика обработки закрытия
    CleanupTabResources(e.TabId);
}

Создание экземпляра события


Guid closedTabId = Guid.NewGuid();
CloseTabEventArgs eventArgs = new CloseTabEventArgs("Документ 1", closedTabId);
Важно:
  • Свойства TabTitle и TabId доступны только для чтения после создания объекта
  • Используется в связке с событиями интерфейса (например, закрытие вкладки в UI)
  • Наследование от MarshalByRefObject позволяет использовать объект в сценариях удаленного взаимодействия

Связь с системой

Интегрирован с:

  • Менеджером вкладок интерфейса
  • Системой событий приложения
  • Механизмами маршалинга при кросс-доменном взаимодействии
