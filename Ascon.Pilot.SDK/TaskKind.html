Перечисление TaskKind

Описание

Определяет типы задач в системе Pilot.SDK. Используется для классификации и фильтрации задач по их назначению.

Значения

Тип задачи Описание
Task Обычная задача (по умолчанию)
Agreement Задача на согласование
Acquaintance Задача на ознакомление

Особенности

  • Используется в связке с SystemTaskAttributes.KIND для фильтрации задач
  • Значение Task является значением по умолчанию при создании новых задач
  • Типы задач влияют на доступные действия и этапы выполнения

Примеры использования

Определение типа задачи


TaskKind taskType = GetTaskKind();
switch (taskType)
{
    case TaskKind.Task:
        ProcessRegularTask();
        break;
    case TaskKind.Agreement:
        StartApprovalProcess();
        break;
    case TaskKind.Acquaintance:
        NotifyUsers();
        break;
}

Фильтрация задач по типу


var agreementTasks = SearchQueryBuilder.Create()
    .Where(SystemTaskAttributes.Kind.EqualTo(TaskKind.Agreement))
    .ToList();
Рекомендации:
  • Используйте TaskKind вместо строковых литералов для повышения type safety
  • Проверяйте тип задачи перед выполнением специфичных операций
  • Учитывайте, что Acquaintance задачи могут иметь ограниченный набор статусов
