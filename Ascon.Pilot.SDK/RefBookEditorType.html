Перечисление RefBookEditorType

Описание

Определяет тип элемента управления для редактирования ссылок на справочники в карточке объекта.

Значения перечисления

Значение Описание Визуальное представление
Dialog Отображает кнопку для выбора значений через диалоговое окно справочника Кнопка с эллипсом (…)
ComboBox Отображает выпадающий список для выбора значения Поле со стрелкой раскрытия списка
Схема выбора типа:
Dialog → Для больших справочников
ComboBox → Для справочников с небольшим количеством элементов

Примеры использования

Настройка поля в карточке объекта


var field = card.AddField("Country");
field.RefBookEditorType = RefBookEditorType.ComboBox;
field.DataSource = countryDictionary;

Динамическая настройка


if (itemsCount > 20)
{
    editorType = RefBookEditorType.Dialog;
}
else
{
    editorType = RefBookEditorType.ComboBox;
}
Важно:
  • ComboBox подходит для справочников до 50 элементов
  • Dialog обеспечивает поиск по большим справочникам
  • Выбор типа влияет на UX-дизайн карточки объекта
Рекомендации:
  • Используйте Dialog для иерархических справочников
  • Для обязательных полей с малым набором значений предпочтителен ComboBox
  • Тип редактора можно комбинировать с атрибутом Required
