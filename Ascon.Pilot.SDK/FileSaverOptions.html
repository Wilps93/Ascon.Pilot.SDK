Класс FileSaverOptions

Описание

Класс для настройки параметров сохранения файлов. Наследуется от MarshalByRefObject для поддержки работы в распределенных средах и сериализации.

Наследование


System.Object
    System.MarshalByRefObject
        FileSaverOptions

Свойства

Свойство Тип Описание
GraphicLayerOption GraphicLayerOption Определяет способ внедрения графических слоев, текстовых меток и штрих-кодов

Примеры использования

Базовая настройка


FileSaverOptions options = new FileSaverOptions
{
    GraphicLayerOption = GraphicLayerOption.EmbedAll
};

Интеграция с сохранением файла


IFileSaver saver = GetFileSaver();
FileSaverOptions options = new FileSaverOptions
{
    GraphicLayerOption = GraphicLayerOption.EmbedTextOnly
};

saver.SaveFile(file, options);
Важно:
  • Свойство GraphicLayerOption требует предварительной настройки в системе
  • Значение по умолчанию: GraphicLayerOption.None (если не указано)
  • Для работы с удаленными объектами используйте MarshalByRefObject возможности
Рекомендации:
  • Проверяйте поддержку выбранных опций в вашей версии SDK
  • Используйте GraphicLayerOption.None для минимального размера файла
  • Для массовых операций кэшируйте настройки
