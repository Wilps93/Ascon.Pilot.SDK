Перечисление State [Устаревшее]

Важно: Этот тип помечен как устаревший. Рекомендуется использовать актуальные версии API и проверять обновления документации.

Описание

Определяет возможные состояния рабочего процесса (Workflow) в системе.

Значения

Состояние Описание
None Черновик - начальное состояние
Assigned Выдано - задача назначена исполнителю
InProgress В работе - задача выполняется
Revoked Отозвано - задача отменена
OnValidation На проверке - ожидает подтверждения
Completed Выполнено - задача завершена

Жизненный цикл

Типичная цепочка переходов:

None → Assigned → InProgress → OnValidation → Completed

Пример использования


State currentState = GetWorkflowState();
switch (currentState)
{
    case State.None:
        // Инициализация
        break;
    case State.Assigned:
        StartProcessing();
        break;
    case State.Completed:
        ArchiveDocument();
        break;
}
Рекомендации:
  • Избегайте использования в новых проектах
  • Проверяйте документацию на наличие обновленных аналогов
  • Не используйте для хранения состояний - перечисление не поддерживает флаги
