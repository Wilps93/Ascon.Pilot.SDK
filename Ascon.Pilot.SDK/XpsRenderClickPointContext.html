Класс XpsRenderClickPointContext

Описание

Контекст для обработки событий кликов в XPS-рендере с привязкой к координатам документа. Наследуется от XpsRenderContext.

Наследование

  • XpsRenderContext
  • → XpsRenderClickPointContext

Конструктор


public XpsRenderClickPointContext(
    IDataObject dataObject,
    DateTime selectedVersion,
    bool isDocumentLoaded,
    Point clickPoint,
    int pageNumber
)
Параметр Описание
dataObject Целевой объект данных, связанный с документом
selectedVersion Временная метка выбранной версии документа
isDocumentLoaded Флаг готовности документа в просмотрщике
clickPoint Координаты клика в системе координат документа
pageNumber Номер страницы, на которой произошел клик

Свойства

Свойство Тип Доступ Описание
ClickPoint Point Только чтение Координаты клика в документе
PageNumber int Чтение/Запись Номер страницы документа

Примеры использования

Создание контекста


var data = GetDocumentData();
var context = new XpsRenderClickPointContext(
    data,
    DateTime.Now,
    isDocumentLoaded: true,
    clickPoint: new Point(150, 200),
    pageNumber: 3
);

Обработка координат


void HandleContextMenu(XpsRenderClickPointContext context)
{
    if (context.IsDocumentLoaded)
    {
        Console.WriteLine($"Клик на странице {context.PageNumber} в точке ({context.ClickPoint.X}, {context.ClickPoint.Y})");
    }
}
Важно:
  • Свойство ClickPoint доступно только для чтения
  • Перед использованием необходимо проверять IsDocumentLoaded
  • Номер страницы начинается с 0 или 1 в зависимости от реализации рендера

Рекомендации

  • Используйте для передачи координат в обработчики контекстного меню
  • Проверяйте актуальность версии документа через SelectedVersion
  • Для сложной логики создавайте наследников этого класса
