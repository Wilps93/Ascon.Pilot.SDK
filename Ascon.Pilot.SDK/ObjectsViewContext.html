Класс ObjectsViewContext

Описание

Контекст представления объектов, используемый для передачи информации о выбранных объектах и режиме отображения. Наследует MarshalByRefObject для поддержки маршалинга в распределенных средах.

Конструктор


public ObjectsViewContext(
    IEnumerable selectedObjects,
    bool isContext
)
Параметр Тип Описание
selectedObjects IEnumerable Коллекция выбранных объектов данных
isContext bool Флаг контекстного режима отображения

Свойства

Свойство Тип Описание
SelectedObjects IEnumerable Выбранные объекты (только для чтения)
IsContext bool Признак контекстного режима (только для чтения)

Особенности

  • Свойства доступны только для чтения после создания объекта
  • Используется для передачи контекста между компонентами системы
  • Наследование от MarshalByRefObject позволяет использовать объект в Remoting

Примеры использования

Создание контекста


var selected = GetSelectedObjects();
var context = new ObjectsViewContext(selected, isContext: true);

Проверка контекстного режима


if (context.IsContext)
{
    Console.WriteLine($"Режим контекста. Выбрано объектов: {context.SelectedObjects.Count()}");
}

Важные замечания

  • Изменение коллекции SelectedObjects после создания контекста не влияет на внутреннее состояние
  • Для корректной работы в распределенных системах требуется сериализация объектов
  • В XML-комментарии класса обнаружена опечатка: "Contex" вместо "Context"

Рекомендации

  • Используйте для передачи данных между слоями приложения
  • Проверяйте IsContext перед выполнением контекстных операций
  • Кэшируйте результаты SelectedObjects при многократном доступе
