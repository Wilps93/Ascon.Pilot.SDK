Класс LoadedEventArgs

Назначение

Аргумент события, уведомляющего о завершении загрузки расширения. Используется для реализации обратных вызовов при инициализации модулей.

Особенности:
  • Наследуется от MarshalByRefObject для поддержки работы в распределенных средах
  • В текущей реализации не содержит дополнительных данных

Иерархия наследования


System.Object 
→ System.MarshalByRefObject 
→ Ascon.Pilot.SDK.LoadedEventArgs

Пример использования


public class ExtensionLoader 
{
    public event EventHandler ExtensionLoaded;

    protected virtual void OnExtensionLoaded()
    {
        ExtensionLoaded?.Invoke(this, new LoadedEventArgs());
    }
}

// Подписка на событие
var loader = new ExtensionLoader();
loader.ExtensionLoaded += (sender, args) => 
{
    Console.WriteLine("Расширение успешно загружено!");
};

Особенности

  • Предназначен для использования в качестве маркерного класса (Marker Class)
  • Может быть расширен в будущих версиях SDK
  • Рекомендуется использовать в паре с EventHandler
Внимание:
  • Не содержит пользовательских данных в текущей реализации
  • Не требует сериализации благодаря наследованию MarshalByRefObject
