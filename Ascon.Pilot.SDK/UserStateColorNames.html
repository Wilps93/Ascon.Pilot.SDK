Перечисление UserStateColorNames

Описание

Определяет набор предустановленных цветов для визуального отображения состояний пользователей в интерфейсе системы.

Значения

Значение Описание
None Отсутствие цветового выделения
Color1 Первый системный цвет (конкретный оттенок определяется темой)
Color2 Второй системный цвет
Color3 Третий системный цвет
Color4 Четвертый системный цвет
Color5 Пятый системный цвет
Color6 Шестой системный цвет
Color7 Седьмой системный цвет

Примеры использования

Установка цвета состояния


public void SetUserColor(UserStateColorNames color)
{
    // Применяем цвет к интерфейсу
    currentColor = color;
    UpdateUserInterface();
}

Проверка текущего цвета


if (user.Color == UserStateColorNames.Color3)
{
    // Логика для третьего цвета
    ApplySpecialFormatting();
}
Важно:
  • Конкретные цветовые оттенки зависят от текущей темы оформления
  • Рекомендуется использовать None для сброса выделения
  • Цвета могут иметь специальное назначение в разных модулях системы

Рекомендации

  • Используйте цвета для визуальной группировки пользователей
  • Проверяйте поддержку цветов в текущей версии темы
  • Для кастомизации цветов обратитесь к документации по стилям
