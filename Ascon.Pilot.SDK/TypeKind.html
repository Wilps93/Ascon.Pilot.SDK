Перечисление TypeKind

Описание

Определяет категории типов объектов в системе. Используется для различия пользовательских и системных сущностей.

Значения

Значение Описание
User Пользовательские типы, создаваемые через интерфейс разработки
System Системные типы, встроенные в платформу и защищенные от изменений

Примеры использования

Проверка типа объекта


TypeKind objectType = GetObjectType();
if (objectType == TypeKind.User)
{
    // Логика для пользовательских типов
    Console.WriteLine("Обработка пользовательского типа");
}

Работа с системными типами


void ProcessType(TypeKind kind)
{
    switch(kind)
    {
        case TypeKind.System:
            // Отключаем редактирование для системных типов
            DisableEditing();
            break;
        default:
            // Стандартная обработка
            EnableEditing();
            break;
    }
}
Важно:
  • Системные типы (System) нельзя модифицировать через API
  • При создании новых типов всегда указывайте User
  • Проверяйте тип перед выполнением операций изменения структуры данных

Рекомендации

  • Используйте проверку типа в методах, работающих с метаданными
  • Для системных типов реализуйте отдельную логику обработки
  • В документации явно указывайте, с какими типами работает каждый метод
