Интерфейс IPilotDialogOptions

Описание

Интерфейс для конфигурации диалоговых окон с использованием паттерна Builder. Позволяет настраивать параметры отображения и поведения диалогов.

Методы конфигурации

Метод Параметры Описание
WithCaption string caption Устанавливает заголовок диалогового окна
WithOkButtonCaption string okButtonCaption Задает текст для кнопки подтверждения
WithParentWindow IntPtr parentWindow Привязывает диалог к родительскому окну (для модальных окон)
WithAllowMultiSelect bool allowMultiSelect Разрешает множественный выбор элементов
WithAllowChecking bool allowChecking Включает/отключает отображение чекбоксов

Примеры использования

Настройка базового диалога


var options = dialogService.CreateOptions()
    .WithCaption("Подтверждение действия")
    .WithOkButtonCaption("Применить")
    .WithAllowMultiSelect(false);

Модальное окно с привязкой к родителю


IntPtr mainWindowHandle = GetMainWindowHandle();
var options = dialogService.CreateOptions()
    .WithParentWindow(mainWindowHandle)
    .WithAllowChecking(true);
Примечание:
  • Все методы возвращают сам интерфейс (fluent interface)
  • Параметры применяются до отображения диалога
  • Родительское окно указывается через хендл (IntPtr)
Рекомендации:
  • Для модальных окон обязательно указывайте родительское окно
  • Проверяйте поддержку множественного выбора в конкретной реализации диалога
