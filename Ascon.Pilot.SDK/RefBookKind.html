Перечисление RefBookKind

Описание

Определяет тип источника данных для справочников в системе. Контролирует, каким образом заполняются значения в справочнике.

Значения перечисления

Значение Описание Источник данных
Enum Статический справочник с фиксированным набором значений Предопределенный список констант
Object Динамический справочник, связанный с объектами системы Дочерние объекты указанного родителя
OrgUnit Справочник организационной структуры Элементы из иерархии организации (подразделения, должности)
Type Справочник типов объектов Метаданные системы (типы документов, классы объектов)

Сценарии использования

Сценарий Рекомендуемый тип
Страны/валюты/статусы Enum
Дерево объектов проекта Object
Подразделения компании OrgUnit
Типы документов в системе Type

Примеры настройки

Создание справочника типа Enum


var statusRefBook = new ReferenceBookConfig
{
    Kind = RefBookKind.Enum,
    Values = new[] { "New", "In Progress", "Completed" }
};

Связь с организационной структурой


var departmentRefBook = new ReferenceBookConfig
{
    Kind = RefBookKind.OrgUnit,
    Source = OrganisationUnitKind.Department
};
Важно:
  • Тип Object требует указания родительского объекта
  • OrgUnit зависит от текущей организационной структуры
  • Изменения в Type отражаются на всей системе
Рекомендации:
  • Для Enum используйте не более 50 значений
  • Тип Type подходит для системных настроек
  • Для OrgUnit настройте фильтры доступа
