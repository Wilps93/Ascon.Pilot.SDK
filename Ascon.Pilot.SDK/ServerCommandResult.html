Перечисление ServerCommandResult

Описание

Определяет статус выполнения серверной команды. Используется для индикации успеха или ошибки при обработке запросов.

Значения перечисления

Значение Описание Индикатор
Success Команда выполнена успешно
Error Произошла ошибка при выполнении команды

Примеры использования

Проверка статуса выполнения


var result = server.ExecuteCommand(command);
if (result == ServerCommandResult.Success)
{
    // Обработка успешного выполнения
    CompleteOperation();
}
else
{
    // Логирование ошибки
    HandleError();
}

Интеграция с ServerCommandRequestResult


var response = commandExecutor.Execute(command);
switch (response.Result)
{
    case ServerCommandResult.Success:
        ProcessData(response.Data);
        break;
    case ServerCommandResult.Error:
        ShowErrorMessage();
        break;
}
Важно:
  • Значение Error требует дополнительной обработки ошибок
  • Для Success данные в ServerCommandRequestResult.Data гарантированно валидны
  • Используйте логирование для фиксации ошибок
Рекомендации:
  • Всегда проверяйте статус перед использованием результатов
  • Для Error используйте try-catch блоки
  • Предоставляйте пользователю понятные сообщения об ошибках
