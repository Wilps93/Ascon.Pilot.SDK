Интерфейс IObjectDialogPreviewModifier

Назначение

Предоставляет функционал для модификации отображения файловых предпросмотров в диалогах объектов. Позволяет задать путь к файлу, который будет использоваться для предварительного просмотра.

Методы

Метод Параметры Возвращаемое значение Описание
SetFilePath path (string) - Путь к файлу предпросмотра IObjectDialogPreviewModifier Устанавливает путь к файлу для предпросмотра.
Поддерживает fluent-интерфейс для цепных вызовов

Пример использования


// Создание модификатора через фабричный метод (пример)
var previewModifier = ObjectModifierFactory.CreatePreviewModifier();

// Установка пути к файлу предпросмотра
previewModifier
    .SetFilePath(@"C:\previews\object_preview.png")
    .Apply(); // Применение изменений

Ключевые особенности

  • Используется для кастомизации отображения объектов в диалоговых окнах
  • Путь к файлу должен быть доступен для чтения приложением
  • Поддерживает относительные и абсолютные пути
  • Работает в связке с другими модификаторами объектов
Важно:
  • Убедитесь в существовании файла по указанному пути
  • Поддерживаемые форматы файлов зависят от реализации предпросмотра
  • Изменения применяются только после вызова метода Apply()
  • Для сброса пути используйте пустую строку или null
