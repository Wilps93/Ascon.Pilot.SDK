Класс GraphicLayerElementConstants

Описание

Контейнер констант для работы с элементами графических слоев. Содержит стандартные идентификаторы, типы контента и версионные метки.

Константы

Константа Значение Описание
GRAPHIC_LAYER_ELEMENT "PILOT_GRAPHIC_LAYER_ELEMENT_" Базовый префикс для идентификаторов графических элементов
GRAPHIC_LAYER_ELEMENT_CONTENT "PILOT_CONTENT_GRAPHIC_LAYER_ELEMENT_" Префикс для контентных идентификаторов графических элементов
XAML "xaml" Тип контента для XAML-данных
BITMAP "bitmap" Тип контента для растровых изображений

Поля

Поле Тип Значение Описание
Version string "2" Версия элементов графического слоя (может изменяться в runtime)

Примеры использования

Формирование идентификатора


string elementId = GraphicLayerElementConstants.GRAPHIC_LAYER_ELEMENT + "SCHEMA_123";
// Результат: "PILOT_GRAPHIC_LAYER_ELEMENT_SCHEMA_123"

Проверка типа контента


if (contentType == GraphicLayerElementConstants.XAML)
{
    // Обработка XAML-контента
    ProcessXamlData(data);
}

Работа с версией


// Проверка поддержки версии
if (GraphicLayerElementConstants.Version == "2")
{
    EnableNewFeatures();
}
Важно:
  • Поле Version доступно для записи - будьте осторожны при модификации
  • Константы содержат служебные префиксы - не изменяйте их вручную
  • Типы контента чувствительны к регистру
Рекомендации:
  • Используйте константы вместо хардкодных строк для обеспечения совместимости
  • Проверяйте версию перед использованием новых возможностей
  • Для кастомных типов контента создавайте собственные константы
