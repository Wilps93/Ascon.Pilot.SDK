Интерфейс IWorkflowChangeBuilder

Внимание!
  • Все методы и свойства помечены как устаревшие ([Obsolete])
  • Рекомендуется проверить наличие актуальных альтернатив в новой версии SDK

Описание

Fluent-интерфейс для модификации рабочих процессов. Позволяет конфигурировать параметры задач в цепочке вызовов.

Методы

Метод Описание
SetTitle(string) Изменяет заголовок задачи
SetDescription(string) Изменяет описание задачи
SetInitiatorAttachments(IEnumerable) Обновляет список вложений инициатора

Свойства

Свойство Тип Описание
WorkflowObject IWorkflowObject Связанный объект рабочего процесса

Пример использования


using (var modifier = GetWorkflowModifier())
{
    // Создаем билдер и настраиваем параметры
    IWorkflowChangeBuilder builder = modifier.EditWorkflow(workflow)
        .SetTitle("Обновленный заголовок")
        .SetDescription("Новое описание задачи")
        .SetInitiatorAttachments(new List { Guid.NewGuid() });

    // Применяем изменения
    modifier.Apply();
}

Рекомендации

Важно:
  • Используйте цепочки вызовов для конфигурации
  • Проверяйте корректность Guid вложений перед передачей
  • Всегда завершайте изменения вызовом Apply() модификатора
  • Для новых проектов ищите альтернативы в актуальной документации SDK
Известные ограничения:
  • Отсутствует метод удаления вложений - передавайте пустой список для очистки
  • Изменения не сохраняются без вызова Apply() родительского модификатора
