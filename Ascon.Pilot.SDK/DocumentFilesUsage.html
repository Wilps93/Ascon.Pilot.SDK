Перечисление DocumentFilesUsage

Описание

Определяет режимы использования контекста файлов документов в системе. Используется в классе DocumentFilesContext для указания сценария работы.

Значения

Значение Описание
CreateNew Режим создания нового документа ECM через диалоговое окно
ObjectsExplorer Режим отображения в обозревателе объектов
CardView Режим просмотра карточки объекта
TemplatesManager Режим управления шаблонами документов

Примеры использования

Определение режима в контексте


DocumentFilesContext context = GetContext();

switch (context.Usage)
{
    case DocumentFilesUsage.CreateNew:
        // Инициализация интерфейса создания документа
        InitializeCreateDialog();
        break;

    case DocumentFilesUsage.CardView:
        // Настройка режима просмотра карточки
        SetupReadOnlyMode();
        break;

    default:
        // Обработка других сценариев
        HandleDefaultCase();
        break;
}

Фильтрация по режиму


if (context.Usage == DocumentFilesUsage.TemplatesManager)
{
    // Отображаем элементы управления шаблонами
    ShowTemplateControls();
}
Важно:
  • Значения перечисления используются в связке с классом DocumentFilesContext
  • Режим ObjectsExplorer часто применяется в деревьях навигации
  • Режим CreateNew требует дополнительной валидации прав доступа
Рекомендации:
  • Используйте switch для обработки разных режимов
  • Проверяйте режим перед выполнением специфичных операций
  • Учитывайте режим при отображении UI-компонентов
