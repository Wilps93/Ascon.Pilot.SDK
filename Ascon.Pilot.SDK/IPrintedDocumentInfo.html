Интерфейс IPrintedDocumentInfo

Описание

Интерфейс предоставляет информацию о документе, подготовленном для печати, включая настройки принтера, поток данных документа и его уникальный идентификатор.

Свойства

Свойство Тип Описание
PrinterSettings IPrinterUserSettings Настройки принтера (формат, ориентация, подключение и т.д.)
DocumentStream Stream Поток данных документа (например, PDF или XPS содержимое)
DocumentId Guid Уникальный идентификатор документа в системе

Примеры использования

Получение информации о документе


IPrintedDocumentInfo documentInfo = GetPrintedDocument();
Guid docId = documentInfo.DocumentId;
using (Stream stream = documentInfo.DocumentStream)
{
    // Работа с потоком документа
    ProcessDocumentStream(stream);
}

Проверка настроек принтера


if (documentInfo.PrinterSettings != null)
{
    // Применение специфичных настроек печати
    ApplyPrinterSettings(documentInfo.PrinterSettings);
}
Важно:
  • Свойства доступны только для чтения (get)
  • DocumentStream должен быть корректно закрыт после использования
  • В интерфейсе опечатка в XML-комментарии: "Summery" вместо "Summary"

Рекомендации

  • Для работы с DocumentStream используйте конструкцию using
  • Проверяйте PrinterSettings на null перед использованием
  • DocumentId может использоваться для отслеживания версий документа
