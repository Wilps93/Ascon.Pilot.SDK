Перечисление FileHandlerSource

Описание

Определяет источники файлов в системе. Поддерживает комбинирование значений через битовые операции ([Flags]).

Значения

Значение Числовой код Описание
Unknown 0 Неизвестный источник (значение по умолчанию)
Publication 1 Файл связан с публикацией
ObjectCreation 2 Файл используется при создании объекта
AutoImport 4 Файл получен через автоматический импорт из папки

Примеры комбинаций


// Объединение источников
FileHandlerSource combined = FileHandlerSource.Publication | FileHandlerSource.AutoImport;

// Проверка наличия флага
if (combined.HasFlag(FileHandlerSource.Publication))
{
    // Обработка публикации
}

// Битовая операция проверки
if ((combined & FileHandlerSource.AutoImport) == FileHandlerSource.AutoImport)
{
    // Обработка автоимпорта
}

Особенности

  • Значение Unknown не участвует в комбинациях (0 не влияет на битовые операции)
  • Для проверки нескольких флагов используйте HasFlag() или битовые операции
  • Числовые коды соответствуют степеням двойки для корректной работы битовых масок
Важно:
  • Не используйте Unknown в комбинациях с другими флагами
  • Проверяйте наличие Unknown отдельно для обработки неопределенных состояний
