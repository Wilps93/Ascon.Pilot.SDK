Класс SystemStates

Описание

Статический класс, содержащий предопределенные идентификаторы и названия системных состояний задач. Используется для унификации работы с состояниями в рамках Pilot.SDK.

Поля

Имя Тип Значение Описание
TASK_REVOKED_STATE_ID Guid abdbe49a-7094-4084-9673-eb5fb3f95262 Идентификатор состояния "Задача отозвана"
TASK_NONE_STATE_ID Guid d8ae8c3a-6f46-45d2-835b-563fe2b47acd Идентификатор начального/пустого состояния задачи
TASK_NO_EXECUTOR_ASSIGNED_STATE_NAME string "noExecutorAssigned" Название состояния "Исполнитель не назначен"

Особенности

  • GUID обеспечивают уникальную идентификацию состояний в распределенных системах
  • Строковые названия состояний чувствительны к регистру
  • Состояние TASK_NONE_STATE_ID часто используется как значение по умолчанию

Примеры использования

Проверка состояния задачи


Guid currentState = task.StateId;
if (currentState == SystemStates.TASK_REVOKED_STATE_ID)
{
    Console.WriteLine("Задача была отозвана");
}

Сравнение названий состояний


string stateName = task.State.Name;
if (stateName.Equals(
    SystemStates.TASK_NO_EXECUTOR_ASSIGNED_STATE_NAME, 
    StringComparison.Ordinal))
{
    AssignExecutor();
}
Рекомендации:
  • Для сравнения строковых состояний используйте StringComparison.Ordinal
  • Проверяйте наличие состояния через .Equals() вместо оператора ==
  • Избегайте прямого создания новых состояний - используйте предопределенные значения
