Интерфейс ITaskBuilder

Внимание! Все методы этого интерфейса помечены как устаревшие (Obsolete). Использование может привести к ошибкам в будущих версиях SDK.

Описание

Флюент-интерфейс для модификации задач (Task). Не рекомендуется к использованию - сохраняется исключительно для поддержки legacy-кода.

Устаревшие методы

Метод Описание Статус
SetAttribute Установка атрибутов задачи (string, int, double и т.д.) Obsolete
SetTitle Изменение заголовка задачи Obsolete
SetDescription Изменение описания задачи Obsolete
SetState Изменение состояния задачи Obsolete
SetExecutor Назначение исполнителя по ID должности Obsolete
SetDeadline Установка абсолютного дедлайна Obsolete
SetRelativeDeadline Установка относительного дедлайна Obsolete
SetInitiatorAttachments Управление вложениями инициатора Obsolete
SetExecutorAttachments Управление вложениями исполнителя Obsolete
SetExecutorRole Установка роли исполнителя Obsolete
SetValidationPhase Включение/отключение фазы валидации Obsolete
SetAuditors Назначение аудиторов Obsolete
SetTags Установка тегов задачи Obsolete
SetKind Установка типа задачи Obsolete
TaskObject Свойство для получения связанной задачи Obsolete

Пример (только для legacy-кода)


// Пример устаревшего кода (не рекомендуется)
var taskBuilder = taskBuilderFactory.Create();
taskBuilder
    .SetTitle("Устаревшая задача")
    .SetDescription("Пример legacy-метода")
    .SetExecutor(123)
    .SetDeadline(DateTime.Now.AddDays(7))
    .SetAttribute("Priority", "High");

var task = taskBuilder.TaskObject;
Рекомендации:
  • Избегайте использования в новом коде
  • Проверьте документацию SDK на наличие современных альтернатив
  • Планируйте миграцию legacy-кода
Важно: Все методы будут удалены в будущих версиях SDK без обратной совместимости.
