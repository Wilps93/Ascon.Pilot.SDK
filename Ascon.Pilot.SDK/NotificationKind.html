Перечисление NotificationKind

Описание

Определяет типы уведомлений о изменениях в системе. Каждое значение соответствует определённому действию с объектами или задачами.

Важные замечания

Устаревшие элементы:
  • Все элементы с Task* помечены как устаревшие
  • ObjectAnnotationChanged заменён на ObjectAnnotationChanged2

Примеры использования

Обработка уведомлений


void ProcessNotification(NotificationKind kind)
{
    switch (kind)
    {
        case NotificationKind.ObjectCreated:
            Console.WriteLine("Новый объект создан");
            break;
            
        case NotificationKind.ObjectAttributeChanged:
            Console.WriteLine("Атрибуты объекта изменены");
            break;
            
        case NotificationKind.Automation:
            Console.WriteLine("Автоматическое уведомление");
            break;
            
        // Обработка устаревших значений
        case NotificationKind.ObjectAnnotationChanged:
            Console.WriteLine("Используйте ObjectAnnotationChanged2");
            break;
    }
}

Проверка устаревших элементов


if (Enum.IsDefined(typeof(NotificationKind), NotificationKind.TaskCreated))
{
    // Предупреждение: Используется устаревший элемент
    Console.WriteLine("Обнаружено устаревшее уведомление о задаче");
}
