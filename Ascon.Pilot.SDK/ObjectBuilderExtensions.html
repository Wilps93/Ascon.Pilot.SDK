Класс ObjectBuilderExtensions

Описание

Статический класс, расширяющий функциональность IObjectBuilder методами для управления файлами объектов.

Методы

RemoveFile

Удаляет файл из текущего снимка объекта по его идентификатору.

Параметр Тип Описание
objectBuilder IObjectBuilder Билдер объекта, с которым выполняется операция
fileId Guid Уникальный идентификатор удаляемого файла

Возвращаемое значение

Возвращает модифицированный экземпляр IObjectBuilder, позволяя использовать метод в цепочке вызовов (fluent interface).

Пример использования


public void RemoveFileExample(IObjectBuilder builder, Guid fileId)
{
    // Удаление файла и сохранение изменений
    builder
        .RemoveFile(fileId)
        .SaveChanges(); // Важно сохранить изменения!
}

Важные замечания

  • Метод модифицирует текущий снимок объекта, но не сохраняет изменения в базе данных
  • Необходимо вызывать SaveChanges() после операции для сохранения результатов
  • Если файл с указанным fileId не существует, метод не генерирует исключение

Рекомендации

  • Проверяйте наличие файла в объекте перед удалением через builder.Files.Contains(fileId)
  • Используйте блок try-catch при работе с критически важными данными
