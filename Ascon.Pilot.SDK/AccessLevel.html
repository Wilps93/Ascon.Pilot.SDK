Перечисление AccessLevel

Описание

Определяет уровни доступа к объектам системы. Перечисление поддерживает комбинации флагов для гибкого управления правами.

Значения

Значение Описание
None Доступ полностью запрещен
Create Право создавать новые объекты
Edit Право редактировать существующие объекты
View Право просматривать объекты
Freeze Право блокировать объекты (например, для предотвращения изменений)
Agreement Право подтверждать или согласовывать объекты
ViewCreate Комбинированный доступ: просмотр + создание
ViewEdit Комбинированный доступ: просмотр + создание + редактирование
ViewEditAgreement Комбинированный доступ: просмотр + создание + редактирование + согласование
Full Полный доступ ко всем действиям

Примеры использования

Проверка прав


AccessLevel userAccess = AccessLevel.ViewEdit;
if ((userAccess & AccessLevel.Edit) == AccessLevel.Edit)
{
    // Разрешено редактирование
}

Комбинирование прав


AccessLevel customAccess = AccessLevel.View | AccessLevel.Agreement;

Проверка на полный доступ


if (userAccess.HasFlag(AccessLevel.Full))
{
    // Пользователь имеет все права
}
Важно: В значении ViewEditAgrement пропущена буква 'e' (корректное название: ViewEditAgreement). Убедитесь, что в коде используется правильное название.
