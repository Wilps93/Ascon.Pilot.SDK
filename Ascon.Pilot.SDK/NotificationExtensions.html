Класс NotificationExtensions

Важно:
  • Статический класс с методами расширения для INotification
  • Использует рефлексию для доступа к внутренним свойствам

Назначение

Предоставляет вспомогательные методы для работы с уведомлениями (INotification), расширяя их функциональность.

Методы

ChangesetId

Получает идентификатор набора изменений, вызвавшего уведомление

Сигнатура


public static long ChangesetId(this INotification notification)

Параметры

notification INotification Объект уведомления

Возвращает

Инкрементный идентификатор набора изменений (long)

Примеры использования

Получение идентификатора изменений


void HandleNotification(INotification notification)
{
    try
    {
        long changesetId = notification.ChangesetId();
        Console.WriteLine($"Обработан набор изменений: {changesetId}");
    }
    catch (Exception ex)
    {
        // Обработка ошибок рефлексии
        Console.WriteLine($"Ошибка: {ex.Message}");
    }
}

Особенности

  • Метод использует рефлексию через Extensions.GetProperty
  • Может вызывать InvalidCastException при некорректном типе свойства
  • Зависит от внутренней реализации объекта уведомления
Внимание:
  • Не гарантирует работоспособность при изменении структуры уведомлений
  • Требует обработки исключений при использовании
  • Не рекомендуется для публичных API
Рекомендации:
  • Используйте только для внутренних нужд системы
  • Проверяйте наличие свойства перед вызовом
  • Кэшируйте результаты для повышения производительности
