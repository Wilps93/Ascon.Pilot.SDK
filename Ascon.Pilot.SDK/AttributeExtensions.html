Класс AttributeExtensions

Описание

Статический класс с методами расширения для интерфейса IAttribute. Предоставляет дополнительные методы для работы с метаданными атрибутов.

Методы расширения

EditableForPositionsSource


public static string[] EditableForPositionsSource(this IAttribute attribute)

Возвращает список позиций (ролей/прав), для которых доступно редактирование атрибута.

Параметр Описание
attribute Объект IAttribute, для которого выполняется проверка

Возвращаемое значение: Массив строк с идентификаторами позиций

Obligatory


public static bool Obligatory(this IAttribute attribute)

Проверяет, является ли атрибут обязательным для заполнения.

Параметр Описание
attribute Объект IAttribute, который проверяется

Возвращаемое значение: true - атрибут обязателен, false - не обязателен

Примеры использования

Проверка обязательности атрибута


IAttribute attribute = GetAttribute();
if (attribute.Obligatory())
{
    // Требуется заполнение
    Console.WriteLine("Атрибут обязателен для заполнения");
}

Получение доступных позиций для редактирования


string[] positions = attribute.EditableForPositionsSource();
foreach (var position in positions)
{
    Console.WriteLine($"Доступ для позиции: {position}");
}
Важно:
  • Методы используют внутренний механизм получения свойств через Extensions.GetProperty
  • Для корректной работы требуется, чтобы в объекте IAttribute были установлены соответствующие метаданные
  • Рекомендуется проверять наличие свойств перед использованием методов
