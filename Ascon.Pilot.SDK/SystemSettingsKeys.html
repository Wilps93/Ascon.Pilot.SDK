Класс SystemSettingsKeys

Описание

Статический класс, содержащий предопределенные ключи системных настроек Pilot.SDK. Каждый ключ представляет собой строку в формате "Название-GUID" для уникальной идентификации функциональных настроек.

Важно: Все ключи доступны только для чтения и не должны изменяться вручную.

Особенности

  • Формат ключей: "Название-xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx"
  • Ключи используются для доступа к настройкам через API конфигурации
  • GUID обеспечивают уникальность и предотвращают конфликты имен

Пример использования


// Получение ключа настроек
string autoPublishKey = SystemSettingsKeys.AutoPublishingExtensionsFeatureKey;

// Использование ключа для доступа к конфигурации
var settings = ConfigurationManager.GetSetting(autoPublishKey);
Предупреждения:
  • В ключе AutoCommitingFileExtensionsFeatureKey присутствует лишний пробел в начале значения
  • Название AgreementRolesFeatureKey повторяется с разными GUID - требует уточнения
  • Избегайте прямого редактирования значений - используйте API конфигурации
