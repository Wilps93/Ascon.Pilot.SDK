Перечисление AttributeType

Описание

Определяет возможные типы атрибутов объектов в системе. Используется для указания типа данных при работе с метаданными и поиском.

Значения

Значение Описание
Integer Целочисленный атрибут (32-битный)
Double Числовой атрибут с плавающей точкой двойной точности
DateTime Атрибут, хранящий дату и время
String Строковый атрибут
Decimal Десятичный числовой атрибут (высокая точность)
Numerator Атрибут для нумераторов (специальный тип для автоинкрементных значений)
Array Массив значений (коллекция элементов)
UserState Атрибут состояния пользователя (идентификатор состояния)
OrgUnit Атрибут организационной единицы (подразделение/отдел)

Примеры использования

Проверка типа атрибута


AttributeType attrType = GetAttributeType();
if (attrType == AttributeType.DateTime)
{
    // Обработка даты
    Console.WriteLine("Обнаружен атрибут типа DateTime");
}

Создание атрибута с указанием типа


IAttribute attribute = factory.CreateAttribute(
    "Price",
    AttributeType.Decimal,
    19.99m
);
Важно:
  • Тип Numerator обычно используется для автоматически генерируемых уникальных номеров
  • UserState и OrgUnit часто связаны с системой прав и ролей
  • Для поиска по атрибутам используйте класс AttributeFields с соответствующими методами

Связь с другими компонентами

Типы из этого перечисления используются в:

  • Методах класса AttributeFields для построения поисковых условий
  • Системе метаданных объектов
  • Механизмах валидации данных
