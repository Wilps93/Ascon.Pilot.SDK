Класс SystemAttributeNames

Описание

Содержит предопределенные имена системных атрибутов для работы с объектами в Pilot.SDK. Каждый атрибут содержит уникальный идентификатор (GUID) для обеспечения целостности данных.

Важно: Некоторые атрибуты помечены как устаревшие ([Obsolete]). Их использование в новом коде не рекомендуется.

Особенности

  • Имена атрибутов содержат уникальные GUID для предотвращения конфликтов
  • Атрибуты с постфиксом _DEPRECATED сохранены для обратной совместимости
  • Используйте SystemAttributeNames вместо строковых литералов для повышения type safety

Примеры использования


// Получение значения атрибута
string deleteDate = object.Attributes[SystemAttributeNames.DELETE_DATE];

// Проверка устаревших атрибутов
if (object.Attributes.ContainsKey(SystemAttributeNames.TASK_EXECUTION_DEPARTMENT))
{
    // Обработка устаревшего атрибута
}
Рекомендации:
  • Избегайте использования устаревших атрибутов в новом коде
  • Для работы с задачами используйте новые API вместо атрибутов Task*
  • Проверяйте актуальность атрибутов в документации последней версии SDK
