Класс ServerCommandRequestResult

Описание

Представляет результат выполнения серверной команды. Содержит данные ответа и статус выполнения.

Иерархия:
System.Object → ServerCommandRequestResult

Конструктор

public ServerCommandRequestResult(byte[] data, ServerCommandResult result)

Инициализирует новый экземпляр класса с указанными данными и статусом.

Свойства

Свойство Тип Описание
Data byte[] Бинарные данные ответа сервера
Result ServerCommandResult Статус выполнения команды

Примеры использования

Получение результата команды


var response = commandExecutor.Execute(command);
if (response.Result == ServerCommandResult.Success)
{
    ProcessData(response.Data);
}

Обработка ошибок


var result = SendCommandToServer();
if (result.Result != ServerCommandResult.Success)
{
    Logger.Error($"Ошибка выполнения: {result.Result}");
}
Важно:
  • Свойства доступны только для чтения
  • Data может быть null при статусе Error
  • Для десериализации Data используйте Serializer
Рекомендации:
  • Всегда проверяйте Result перед использованием Data
  • Для критических ошибок используйте ServerCommandResult.Fatal
  • При сериализации учитывайте формат данных сервера
