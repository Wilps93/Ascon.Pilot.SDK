Класс AttributeValueChangedEventArgs

Описание

Класс, предоставляющий данные о изменении значения атрибута объекта. Наследуется от MarshalByRefObject для поддержки маршалинга в распределенных средах.

Наследование

  • System.Object
  • System.MarshalByRefObject
  • AttributeValueChangedEventArgs

Конструктор


public AttributeValueChangedEventArgs(
    object oldValue,
    object newValue,
    ObjectCardContext context
)

Инициализирует новый экземпляр класса.

Параметр Описание
oldValue Предыдущее значение атрибута
newValue Новое значение атрибута
context Контекст объекта, связанного с изменением

Свойства

Имя Тип Описание
OldValue object Предыдущее значение атрибута (только для чтения)
NewValue object Новое значение атрибута (только для чтения)
Context ObjectCardContext Контекст объекта (только для чтения)

Особенности

  • Объекты неизменяемы после создания
  • Поддерживает передачу между доменами приложений
  • Используется в событиях изменения атрибутов

Пример использования


// Обработчик события
void OnAttributeValueChanged(object sender, AttributeValueChangedEventArgs e)
{
    Console.WriteLine($"Старое значение: {e.OldValue}");
    Console.WriteLine($"Новое значение: {e.NewValue}");
    Console.WriteLine($"Контекст объекта: {e.Context.Id}");
}

// Создание экземпляра
var eventArgs = new AttributeValueChangedEventArgs(
    oldValue: 42,
    newValue: 100,
    context: currentContext
);
Важно: Для корректной работы в распределенных системах убедитесь, что типы OldValue/NewValue и ObjectCardContext поддерживают сериализацию.
