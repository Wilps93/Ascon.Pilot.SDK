Интерфейс IAttributeModifier

Описание

Интерфейс для модификации атрибутов объектной карточки. Предоставляет набор методов для установки значений различных типов.

Методы

Метод Описание
SetValue(string name, string value) Устанавливает строковое значение атрибута
SetValue(string name, long value) Устанавливает целочисленное значение (64-bit)
SetValue(string name, double value) Устанавливает значение с плавающей точкой
SetValue(string name, DateTime value) Устанавливает дату/время
SetValue(string name, decimal value) Устанавливает десятичное значение
SetValue(string name, Guid value) Устанавливает уникальный идентификатор
SetValue(string name, int[] value) Устанавливает массив целых чисел (32-bit)
SetValue(string name, Guid[] value) Устанавливает массив уникальных идентификаторов

Особенности

  • Поддерживает строгую типизацию для различных типов данных
  • Работает с простыми типами и массивами
  • Позволяет модифицировать атрибуты в объектной карточке

Примеры использования

Установка простых типов


public void ModifyAttributes(IAttributeModifier modifier)
{
    modifier.SetValue("Name", "Новое название");
    modifier.SetValue("Quantity", 100L);
    modifier.SetValue("Price", 99.95m);
    modifier.SetValue("ReleaseDate", DateTime.Now);
    modifier.SetValue("UniqueId", Guid.NewGuid());
}

Работа с массивами


var ids = new Guid[] 
{ 
    Guid.NewGuid(), 
    Guid.NewGuid() 
};

modifier.SetValue("RelatedIds", ids);
Важно: В оригинальных комментариях к методам для массивов (int[] и Guid[]) указано описание "New decimal value". Это техническая опечатка, реальные типы соответствуют сигнатуре метода.
Рекомендации:
  • Проверяйте типы данных перед установкой значений
  • Для массивов используйте только одномерные массивы
  • Убедитесь в корректности имен атрибутов
