Интерфейс IToolbarToggleButtonItemBuilder

Описание

Представляет переключаемую кнопку (toggle button) на панели инструментов. Наследует функционал базового билдера кнопок и добавляет поддержку состояния "включено/выключено".

Наследование

public interface IToolbarToggleButtonItemBuilder : IToolbarButtonItemBuilder

Наследует все методы из IToolbarButtonItemBuilder для базовой конфигурации кнопки.

Методы

IToolbarToggleButtonItemBuilder WithIsChecked(bool value)

Устанавливает начальное состояние переключателя

  • value - true для включенного состояния, false для выключенного
  • Возвращает текущий экземпляр билдера для цепочки вызовов

Пример использования


// Создание переключаемой кнопки
toolbarBuilder.AddToggleButtonItem("filter_toggle", 0)
    .WithHeader("Фильтр")
    .WithIcon(File.ReadAllBytes("filter.svg"))
    .WithIsChecked(true)
    .WithHint("Включить/выключить фильтрацию");
Важные замечания:
  • Состояние переключателя сохраняется между сессиями только при явной реализации сохранения состояния
  • Для обработки переключений необходимо реализовать соответствующую логику в обработчике команды кнопки
  • Визуальное отображение состояния зависит от реализации UI-фреймворка
  • Метод WithIsChecked() устанавливает только начальное состояние

Связь с IToolbarButtonItemBuilder

Наследует все методы базового билдера:

  • WithHeader()
  • WithIcon()
  • WithIsEnabled()
  • WithShowHeader()
  • WithHint()

Рекомендации

  • Используйте для функций, требующих постоянного включения/выключения (фильтры, режимы отображения)
  • Синхронизируйте состояние с моделью данных приложения
  • Добавляйте визуальные индикаторы состояния через иконки или цвет
  • Для групп переключателей реализуйте логику взаимоисключения
