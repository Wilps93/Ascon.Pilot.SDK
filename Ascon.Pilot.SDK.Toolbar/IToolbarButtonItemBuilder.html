Интерфейс IToolbarButtonItemBuilder

Описание

Предоставляет методы для конфигурации кнопок на панели инструментов. Использует паттерн "строитель" с поддержкой цепочек вызовов (fluent interface).

Методы конфигурации

IToolbarButtonItemBuilder WithHeader(string header)

Устанавливает отображаемый заголовок кнопки

  • header - Текст для отображения
IToolbarButtonItemBuilder WithIcon(byte[] svgIcon)

Задает иконку кнопки в формате SVG

  • svgIcon - Массив байт с SVG-изображением
IToolbarButtonItemBuilder WithIsEnabled(bool value)

Включает/отключает интерактивность кнопки

  • value - true для активной кнопки, false для заблокированной
IToolbarButtonItemBuilder WithShowHeader(bool value)

Управляет отображением текстового заголовка

  • value - true для показа заголовка, false для скрытия
IToolbarButtonItemBuilder WithHint(string hint)

Устанавливает всплывающую подсказку для кнопки

  • hint - Текст подсказки

Пример использования


// Создание кнопки с полной конфигурацией
toolbarBuilder.AddButtonItem("refresh_button", 0)
    .WithHeader("Обновить")
    .WithIcon(File.ReadAllBytes("refresh.svg"))
    .WithIsEnabled(true)
    .WithShowHeader(false)
    .WithHint("Обновить данные");
Важные замечания:
  • SVG-иконки должны быть валидными XML-документами
  • Порядок вызова методов не влияет на результат
  • Значения по умолчанию: заголовок - пустая строка, иконка - отсутствует, кнопка включена, заголовок показывается
  • Для отключения отображения заголовка используйте WithShowHeader(false)

Рекомендации

  • Используйте SVG-иконки оптимизированного размера для улучшения производительности
  • Для динамического обновления состояния используйте паттерн наблюдатель (Observer)
  • Проверяйте наличие иконки перед установкой: if(File.Exists(path)) { ... }
  • Группируйте связанные кнопки визуально через единый стиль иконок
