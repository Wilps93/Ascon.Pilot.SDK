Перечисление TaskTemplateKind [Устаревший]

Описание

Определяет типы шаблонов задач. Перечисление признано устаревшим - рекомендуется использовать современные альтернативы для новых разработок.

Значения

Значение Описание
Common Общий шаблон задачи (для группового использования)
Personal Личный шаблон задачи (для индивидуального использования)

Примеры использования


// Создание шаблона задачи с типом Common
var commonTemplate = new TaskTemplate(
    Guid.NewGuid(),
    "General Approval",
    workflow,
    TaskTemplateKind.Common
);

// Проверка типа шаблона
if (template.Kind == TaskTemplateKind.Personal)
{
    // Логика для личных шаблонов
}
Важно:
  • Перечисление помечено как [Obsolete] - использование в новых проектах не рекомендуется
  • Значение Personal требует дополнительной настройки прав доступа
  • Рекомендуется использовать ITaskTypeDefinition в новых реализациях

Особенности реализации

  • Используется в связке с устаревшим классом TaskTemplate
  • Не содержит методов расширения или дополнительной функциональности
  • Значения не имеют связанных атрибутов или метаданных
