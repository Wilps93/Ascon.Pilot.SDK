Класс AutosignOptions [Устаревший]

Описание

Конфигурация автоматической подписи задач. Класс признан устаревшим - рекомендуется использовать новые механизмы управления подписями.

Поддерживает сериализацию для сохранения состояния.

Свойства

Свойство Тип Описание
Autosign bool Включена ли автоматическая подпись
Role string Роль, связанная с автоматической подписью

Методы

Переопределенные методы

Equals(AutosignOptions) Сравнивает текущий объект с другим
Equals(object) Сравнивает с произвольным объектом
GetHashCode() Возвращает хеш-код объекта

Примеры использования


// Создание конфигурации
var options = new AutosignOptions
{
    Autosign = true,
    Role = "admin"
};

// Сравнение конфигураций
AutosignOptions another = GetOptions();
bool areEqual = options.Equals(another);

// Использование в словаре
Dictionary configs = new Dictionary();
configs.Add(options, "Default config");
Важно:
  • Класс помечен как [Obsolete] - использование в новых проектах не рекомендуется
  • Сравнение учитывает регистр строки Role
  • Хеш-код зависит от значений Autosign и Role
  • При сериализации сохраняются все свойства

Особенности реализации

  • Сравнение объектов происходит по всем свойствам
  • Хеш-код использует комбинацию булева значения и строки
  • Реализация учитывает возможное наследование через проверку GetType()
