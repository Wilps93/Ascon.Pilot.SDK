Класс TaskTemplate [Устаревший]

Описание

Представляет шаблон задачи с метаданными и рабочим процессом. Класс признан устаревшим - рекомендуется использовать современные альтернативы для новых разработок.

Содержит неизменяемые данные о шаблоне задачи.

Конструктор


public TaskTemplate(
    Guid id,
    string name,
    Workflow workflow,
    TaskTemplateKind kind
)

Параметры

id Уникальный идентификатор шаблона
name Название шаблона
workflow Рабочий процесс шаблона (тип Workflow)
kind Тип шаблона задачи

Свойства

Свойство Тип Описание
Id Guid Уникальный идентификатор шаблона (только для чтения)
Name string Название шаблона (только для чтения)
Workflow Workflow Рабочий процесс шаблона (только для чтения)
Kind TaskTemplateKind Тип шаблона задачи (только для чтения)

Примеры использования


// Создание шаблона задачи
var workflow = new Workflow(...);
var template = new TaskTemplate(
    Guid.NewGuid(),
    "Document approval",
    workflow,
    TaskTemplateKind.BusinessProcess
);

// Получение информации о шаблоне
Console.WriteLine($"Template ID: {template.Id}");
Console.WriteLine($"Workflow steps: {template.Workflow.Steps.Count()}");
Важно:
  • Класс помечен как [Obsolete] - использование в новых проектах не рекомендуется
  • Все свойства доступны только для чтения после создания
  • Используемый класс Workflow также считается устаревшим
  • Рекомендуется использовать ITaskTemplateV2 в новых реализациях

Особенности реализации

  • Объект становится неизменяемым после создания
  • Не содержит методов для модификации состояния
  • Зависит от устаревших компонентов (Workflow, Stage)
