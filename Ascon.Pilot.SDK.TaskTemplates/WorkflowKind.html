Перечисление WorkflowKind [Устаревший]

Важно: Это перечисление помечено как устаревшее ([Obsolete]). Рекомендуется использовать альтернативные подходы из текущей версии SDK.

Описание

Определяет тип рабочего процесса (workflow). Используется в классе Workflow для указания категории процесса.

Значения

Имя Описание
Regular Обычный рабочий процесс (базовый тип)
Agreement Процесс, связанный с согласованием документов
Review Процесс, связанный с проверкой/ревизией

Использование в классе Workflow


public class Workflow
{
    public WorkflowKind Kind { get; set; }
    // ... остальные члены класса
}

Пример кода


var workflow = new Workflow
{
    Kind = WorkflowKind.Agreement,
    Title = "Согласование договора"
};

if (workflow.Kind == WorkflowKind.Agreement)
{
    // Логика для процессов согласования
    Console.WriteLine("Обработка согласования");
}
Важные замечания:
  • Значение этого перечисления участвует в вычислении хэш-кода и сравнении объектов Workflow
  • При удалении/изменении элементов перечисления возможна поломка логики сравнения
  • Отсутствие XML-комментариев в исходном коде затрудняет точную интерпретацию значений
