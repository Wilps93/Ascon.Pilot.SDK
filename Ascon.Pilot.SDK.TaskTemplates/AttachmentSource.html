Перечисление AttachmentSource [Устаревший]

Описание

Определяет источник вложений в задачах. Перечисление признано устаревшим - рекомендуется использовать современные альтернативы для новых разработок.

Значения

Значение Описание
Object Вложение представляет собой обычный объект системы
PilotStorage Вложение хранится в специализированном хранилище Pilot Storage

Примеры использования


// Проверка типа вложения
AttachmentSource source = GetAttachmentSource();

if (source == AttachmentSource.PilotStorage)
{
    // Обработка вложений из Pilot Storage
}
else
{
    // Работа с обычными объектами
}

// Использование в условиях фильтрации
IEnumerable FilterAttachments(IEnumerable attachments)
{
    return attachments.Where(a => a.Source == AttachmentSource.Object);
}
Важно:
  • Перечисление помечено как [Obsolete] - использование в новых проектах не рекомендуется
  • Значение PilotStorage требует специфической обработки хранилищем
  • При переходе на новые версии SDK потребуется рефакторинг кода
  • Рекомендуется избегать хранения значений этого типа в долгоживущих данных

Совместимость

Используется в связке с устаревшим классом Attachment. Для новых разработок:

  • Используйте IStorageReference для работы с внешними хранилищами
  • Применяйте ObjectReference для ссылок на объекты системы
