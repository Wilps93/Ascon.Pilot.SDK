Класс Participant [Устаревший]

Описание

Базовый класс для участников задач. Класс признан устаревшим - рекомендуется использовать современные альтернативы для новых разработок.

Содержит базовые свойства и методы для идентификации участников в системе.

Свойства

Свойство Тип Описание
PositionId int Идентификатор позиции участника в системе

Методы

Переопределенные методы

Equals(Participant) Сравнивает текущий объект с другим участником
Equals(object) Сравнивает с произвольным объектом
GetHashCode() Возвращает хеш-код объекта

Примеры использования


// Создание участника
var participant = new Participant
{
    PositionId = 101
};

// Сравнение участников
Participant another = GetParticipant();
bool areEqual = participant.Equals(another);

// Использование в словаре
Dictionary participants = new Dictionary();
participants.Add(participant, "Main participant");
Важно:
  • Класс помечен как [Obsolete] - использование в новых проектах не рекомендуется
  • Сравнение и хеширование основаны исключительно на PositionId
  • Производные классы (например, Executor) также считаются устаревшими
  • Рекомендуется использовать IParticipantReference в новых разработках

Особенности реализации

  • Хеш-код напрямую возвращает значение PositionId
  • Реализация Equals требует строгого совпадения типов
  • Отсутствует виртуальный метод Equals для наследников
