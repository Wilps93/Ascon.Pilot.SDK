Класс AutomationException

Общее описание

Класс исключения AutomationException предназначен для обработки ошибок, возникающих в процессе выполнения автоматизированных операций в системе. Наследуется от стандартного класса System.Exception.

Используется для:

  • Обработки ошибок в рабочих процессах
  • Уведомления о сбоях в автоматизации
  • Логирования критических ситуаций
  • Перехвата исключений в обработчиках событий

Конструкторы

Конструктор Описание
AutomationException() Инициализирует новый экземпляр исключения без текстового сообщения
AutomationException(string message) Инициализирует экземпляр с указанным текстовым сообщением об ошибке

Примеры использования

Обработка исключения в автоматизации


try
{
    // Операция автоматизации
    ProcessAutomationTask();
}
catch (AutomationException ex)
{
    Logger.LogError($"Ошибка автоматизации: {ex.Message}");
    // Обработка специфической ошибки
}

Генерация исключения


if (!IsValidConfiguration(config))
{
    throw new AutomationException("Неверная конфигурация автоматизации");
}

Перехват в цепочке вызовов


public void ExecuteWorkflow()
{
    try
    {
        StartProcess();
        ExecuteSteps();
    }
    catch (AutomationException)
    {
        // Специфическая обработка
        throw; // Проброс исключения
    }
}
Рекомендации:
  • Используйте этот класс только для ошибок автоматизации
  • Добавляйте детальные сообщения для упрощения диагностики
  • Не перехватывайте AutomationException без последующей обработки
  • Дополняйте исключения информацией о контексте выполнения
