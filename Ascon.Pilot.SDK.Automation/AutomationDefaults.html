Класс AutomationDefaults

Общее описание

Статический класс AutomationDefaults содержит набор предопределенных констант для конфигурации автоматизированных процессов в системе. Используется для:

  • Настройки рабочих процессов
  • Конфигурации прав доступа
  • Управления уведомлениями
  • Работы с состояниями объектов
  • Интеграции с внешними сервисами

Примеры использования

Конфигурация SMTP


var smtpConfig = new Dictionary<string, object>
{
    { AutomationDefaults.Host, "smtp.example.com" },
    { AutomationDefaults.Port, 587 },
    { AutomationDefaults.EnableSsl, true },
    { AutomationDefaults.UserName, "user@example.com" },
    { AutomationDefaults.Password, "secure_password" }
};

Настройка перехода состояний


var stateConfig = new TransitionConfiguration
{
    Parameters = 
    {
        { AutomationDefaults.FromStateName, "Draft" },
        { AutomationDefaults.ToStateName, "Published" },
        { AutomationDefaults.AccessLevel, AccessLevel.ViewEdit }
    }
};

Фильтрация изменений


if (changeData.Contains(AutomationDefaults.ItemChangeKind) 
    && changeData[AutomationDefaults.ItemChangeKind] == ChangeKind.Modification)
{
    // Обработка модификаций
}
Примечание: Все константы предназначены только для чтения. Для изменения поведения автоматизации используйте их в конфигурационных словарях.
