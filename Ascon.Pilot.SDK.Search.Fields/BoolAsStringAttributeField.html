Класс BoolAsStringAttributeField

Описание

Класс для работы с булевыми полями, хранящимися в виде строковых значений ("True"/"False") в поисковых запросах. Наследуется от BoolField.

Наследование

  • System.Object
  • BoolField
  • BoolAsStringAttributeField

Конструктор


internal BoolAsStringAttributeField(string fieldName)

Инициализирует новый экземпляр класса с указанием имени поля.

Методы

Be(bool value)


public override ISearchTerm Be(bool value)

Создает поисковый термин для булева значения, преобразованного в строку ("True" или "False").

Особенности реализации

  • Булевы значения преобразуются: true → "True", false → "False"
  • Использует SimpleSearchTerm для формирования условия
  • Предназначен для полей, где логические значения хранятся в текстовом формате

Пример использования


// Создание условия поиска для строкового булева поля
var field = new BoolAsStringAttributeField("IsApproved");
ISearchTerm term = field.Be(true);

// Результат: условие "IsApproved: True"
Важно: Класс имеет внутренний модификатор доступа (internal) и должен использоваться только внутри SDK. Для создания условий используйте публичные методы API.
Рекомендации:
  • Учитывайте регистр значений при работе с разными СУБД
  • Для числовых представлений булевых значений используйте BoolAsInt32Field или BoolAsInt64Field
