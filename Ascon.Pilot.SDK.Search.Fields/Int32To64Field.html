Класс Int32To64Field

Общее описание

Внутренний класс для работы с 32-битными целочисленными полями, преобразованными для хранения в 64-битном формате. Добавляет специальный префикс к значениям для корректной обработки в поисковых запросах.

Иерархия наследования

  • FieldBase
  • → Int32Field
  • → Int32To64Field

Конструктор


internal Int32To64Field(string fieldName)

Инициализирует экземпляр с экранированным именем поля.

Особенности: Доступен только внутри текущей сборки.

Методы

WithPrefix (переопределенный)


protected override string WithPrefix(int value)

Формирует строковое представление числа с префиксом для поискового индекса.

Преобразование: 123 → "@123"

Примеры использования

Преобразование значения


var field = new Int32To64Field("numeric_field");
string result = field.WithPrefix(456);
// Результат: "@456"

Интеграция с поисковыми терминами


ISearchTerm term = field.Be(789);
// Генерирует термин: "numeric_field":"@789"
Важно:
  • Класс предназначен для внутреннего использования в SDK
  • Префикс "@" обеспечивает корректную обработку 32-битных чисел в 64-битных индексах
  • Для внешних запросов используйте публичные классы вроде Int32Field

Особенности

  • Автоматическое добавление специального префикса к числовым значениям
  • Наследует базовую функциональность от Int32Field
  • Обеспечивает совместимость с 64-битными хранилищами данных
