Класс CommandDescription

Общее описание

Класс CommandDescription представляет описание команды в системе. Наследуется от MarshalByRefObject для поддержки работы в распределенных средах. Используется для:

  • Хранения идентификатора команды
  • Сохранения человекочитаемого описания
  • Интеграции с системой команд и операций

Является неизменяемым объектом после создания.

Конструктор

Метод Описание
CommandDescription(Guid commandId, string description) Инициализирует новый экземпляр
Параметры:
  • commandId - Уникальный идентификатор команды
  • description - Текстовое описание команды

Свойства

Свойство Тип Описание
CommandId Guid Уникальный идентификатор команды (только для чтения)
Description string Человекочитаемое описание команды (только для чтения)

Примеры использования

Создание описания команды


var commandId = Guid.NewGuid();
var description = new CommandDescription(
    commandId, 
    "Команда экспорта данных в формате PDF"
);

Console.WriteLine($"ID: {description.CommandId}");
Console.WriteLine($"Описание: {description.Description}");

Использование в реестре команд


var commandRegistry = new Dictionary();
commandRegistry.Add(
    Guid.Parse("123e4567-e89b-12d3-a456-426614174000"),
    new CommandDescription(
        Guid.Parse("123e4567-e89b-12d3-a456-426614174000"),
        "Системная команда сохранения"
    )
);
Важно:
  • Объект становится неизменяемым после создания
  • Используйте уникальные идентификаторы для корректной работы системы
  • Описание должно быть локализовано для пользовательского интерфейса
  • Наследование от MarshalByRefObject требует осторожности в распределенных средах
