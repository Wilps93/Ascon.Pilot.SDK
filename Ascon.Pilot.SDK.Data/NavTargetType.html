Перечисление NavTargetType

Описание

Определяет типы навигационных целей в системе. Используется в связке с интерфейсом INavigationTarget для указания типа объекта, к которому осуществляется переход.

Значения

Значение Описание
None Неопределенный тип (используется как значение по умолчанию)
Object Базовый объект системы
Task Задача или элемент рабочего процесса
ProjectFile Файл проекта
Report Отчетная форма
Chat Чат или обсуждение
DraftSearchTab Вкладка поиска черновиков
XpsMergerTab Вкладка объединения XPS-документов
DocumentComparison Сравнение документов
ObjectRelatedChat Чат, связанный с конкретным объектом
TaskRelatedChat Чат, связанный с конкретной задачей

Пример использования


void NavigateToTarget(INavigationTarget target)
{
    switch (target.Type)
    {
        case NavTargetType.Object:
            OpenObject(target.ObjectId);
            break;
            
        case NavTargetType.Task:
            ShowTaskDetails(target.ObjectId);
            break;
            
        case NavTargetType.Report:
            GenerateReport(target.ObjectId);
            break;
            
        default:
            // Обработка неизвестного типа
            throw new NotSupportedException($"Тип {target.Type} не поддерживается");
    }
}

Особенности

  • Значение None используется для инициализации и обозначения отсутствия типа
  • Типы ObjectRelatedChat и TaskRelatedChat требуют обязательного указания ObjectId
  • Специфичные типы вроде XpsMergerTab могут требовать дополнительных аргументов в INavigationTarget.Args
Важно: При обработке навигации всегда проверяйте поддержку типа через Enum.IsDefined(typeof(NavTargetType), value)
