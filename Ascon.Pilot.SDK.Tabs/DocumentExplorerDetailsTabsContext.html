Класс DocumentExplorerDetailsTabsContext

Описание

Представляет контекст данных для вкладок детализации в Document Explorer. Содержит информацию о выбранном объекте и версии документа.

Наследование

MarshalByRefObject → DocumentExplorerDetailsTabsContext

Конструктор


public DocumentExplorerDetailsTabsContext(
    IDataObject selectedObject,
    DateTime? snapshotDateTime
)
Параметр Описание
selectedObject Выбранный объект данных, для которого отображаются детали
snapshotDateTime Дата и время версии документа. Null для актуальной версии

Свойства

Свойство Тип Описание
SelectedObject IDataObject Выбранный объект данных (только для чтения)
SnapshotDateTime DateTime? Дата версии документа или null для текущей версии (только для чтения)

Особенности

  • Используется для передачи контекста между вкладками детализации Document Explorer
  • Наследование от MarshalByRefObject позволяет использовать объект в сценариях с удаленным взаимодействием
  • Свойства доступны только для чтения после инициализации

Примеры использования

Создание контекста


IDataObject selectedDocument = ...; // Получение объекта документа
DateTime versionDate = new DateTime(2023, 10, 1);

var context = new DocumentExplorerDetailsTabsContext(
    selectedDocument,
    versionDate
);

Проверка версии документа


if (context.SnapshotDateTime.HasValue)
{
    // Работа с архивной версией
    DateTime version = context.SnapshotDateTime.Value;
}
else
{
    // Работа с актуальной версией документа
}
Важно:
  • Изменение состояния объекта после создания невозможно - все свойства доступны только для чтения
  • Для корректной работы с версиями всегда проверяйте значение SnapshotDateTime
  • Класс предназначен для использования внутри системы Document Explorer и не должен создаваться вручную
