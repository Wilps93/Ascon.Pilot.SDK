Статический класс FieldNames

Описание

Контейнер констант с именами полей для построения поисковых запросов в системе. Содержит предопределенные имена системных полей объектов.

Константы

Имя Значение Описание
ID "DObject.Id" Уникальный идентификатор объекта
TYPE_ID "DObject.TypeId" Идентификатор типа объекта
PARENT_ID "DObject.ParentId" Идентификатор родительского объекта
CREATOR_ID "DObject.CreatorId" ID создателя объекта
CREATED "DObject.Created" Дата создания объекта
IS_SECRET "DObject.IsSecret" Флаг конфиденциальности
IS_DELETED "DObject.IsDeleted" Флаг удаления объекта
IS_IN_RECYCLE_BIN "DObject.IsInRecycleBin" Флаг нахождения в корзине
ALL_TEXT "DObject.AllText" Полнотекстовый поиск по содержимому
ALL_SNAPSHOTS_REASON "Dobject.SnapshotsReason" Причина создания снимков объекта
SNAPSHOTS_CREATED "Dobject.SnapshotsCreated" Дата создания снимков
OBJECT_STATE "DObject.State.State" Текущее состояние объекта
STATE_PERSON_ID "DObject.State.PersonId" ID пользователя, изменившего состояние
STATE_DATE "DObject.State.Date" Дата изменения состояния
STATE_POSITION "DObject.State.PositionId" Должность пользователя при изменении состояния
CONTEXT "DObject.Context" Контекст объекта
LEVEL "DObject.Level" Уровень вложенности объекта

Особенности

  • Класс предназначен только для внутреннего использования в SDK
  • Имена полей соответствуют внутренней структуре данных системы
  • Используется для формирования критериев поиска в ISearchService

Примеры использования

Фильтрация по типу объекта


var query = searchService.CreateQuery();
query.AddCondition(FieldNames.TYPE_ID, "contract");

Проверка нахождения в корзине


query.AddCondition(
    FieldNames.IS_IN_RECYCLE_BIN, 
    true
);

Полнотекстовый поиск


query.AddCondition(
    FieldNames.ALL_TEXT, 
    "важный документ"
);
Важно: В константах ALL_SNAPSHOTS_REASON и SNAPSHOTS_CREATED используется написание "Dobject" с маленькой буквы. Это может быть либо особенностью системы, либо опечаткой в названии. Рекомендуется проверять актуальность имен через рефлексию или документацию API.
